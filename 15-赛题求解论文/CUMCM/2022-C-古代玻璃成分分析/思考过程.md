## 1. 题目理解：这道题到底在问什么

先通读题目原文（`/Users/ephemeral/美赛/13-国赛CUMCM/真题2022/2022_C题.md`），拆解出几个关键事实和约束：

1. **背景**：古代玻璃分"高钾玻璃"和"铅钡玻璃"两大类，助熔剂不同（草木灰/硝石类 vs 铅矿石），因此氧化钾(K2O)、氧化铅(PbO)、氧化钡(BaO)天然是化学上的"指纹"成分。
2. **风化是核心干扰变量**：文物长期埋藏，表面与环境发生元素交换，风化会显著改变检测到的成分比例，可能掩盖真实类型。这意味着**问题2、3、4如果直接用风化后的原始检测值做分类/聚类/相关分析，本质上是在给"埋藏环境的化学噪声"建模，而不是给"玻璃配方"建模**——这是我做建模路线选择时最先意识到的一个关键点。
3. **数据质量声明是题目自带的**："成分比例累加和应为100%，但检测误差可能导致偏离，本题规定85%~105%为有效数据"——这是一个明确的、必须先做的数据清洗步骤，不能跳过。
4. 四个问题分别是：
   - 问题1：风化 与 类型/纹饰/颜色 的关系；分类型统计风化前后成分变化规律；**用风化点数据反推风化前成分**（这是一个"求逆"问题，本质上是给风化过程建立一个可逆的变换）。
   - 问题2：无监督/半监督地"发现"高钾-铅钡分类规律（其实标签已知，所以这里的"规律"更多是要求解释清楚"为什么能分开、用什么变量分开"），并在每一大类内部做亚类划分（真正的无监督聚类问题）。
   - 问题3：把问题2/3的分类规则应用到表单3的未知样本上，是一个标准的有监督分类预测任务。
   - 问题4：相关性分析 + 类间比较，是描述性统计+假设检验问题。

**我对题目的核心判断**：这四问不是互相独立的四个小题，而是一条流水线——问题1的"去风化"建模结果，应该是问题2/3/4的输入基础。如果问题2直接对风化后的原始数据聚类，聚出来的"亚类"可能只是"风化程度不同"的伪聚类，而不是"配方不同"的真聚类。这个判断直接决定了我后面统一构建了一张"文物级去风化后代表成分表"，供问题2-4共用。

## 2. 候选方案对比

### 2.1 问题1：风化前成分反推

**候选方案A（个体级回归/机器学习）**：用少量"同一文物既有风化点又有未风化点"的配对数据，训练一个回归模型（每个成分一个线性回归，或多输出模型），学习"风化后→风化前"的映射。
- 实际检查后发现：全部69个采样点中，真正在**同一件文物上**同时存在"常规点"和"未风化点/严重风化点"标注的配对，只有5例（文物8、26、54为"常规点 vs 严重风化点"；文物49、50为"常规点 vs 未风化点"），且大部分标注为"未风化点"的文物（23、25、28、29、42、44、53）只单独测了未风化点，没有配对的风化点数据。
- 样本量太小（n=2 用于铅钡型的真配对），无法支撑一个可信的回归模型，容易过拟合，而且高钾型完全没有配对样本，此路线对高钾型不可行。
- **弃用**，但保留这5个真配对样本用作**验证集**（这是最终采用的做法）。

**候选方案B（组均值比例反演，最终采用）**：分类型（高钾/铅钡）比较"无风化组"与"风化组"（用常规采样点，剔除未风化点/严重风化点这类特殊标注点）逐成分的均值，得到比例系数 k_c = 无风化组均值 / 风化组均值；对每个风化文物的实测成分，乘以对应类型、对应成分的 k_c，再归一化至100%，作为"风化前成分"的估计。
- 优点：只需要组一级的统计量，不依赖罕见的个体配对数据；物理解释清晰（风化导致某些易溶成分流失、另一些成分相对富集或被环境介质带入，组均值比例近似刻画了这种整体趋势）；可以直接用方案A预留的5个真配对样本做**留出验证**，检验组均值方法是否可信。
- 缺点：是一个"平均效应"模型，无法刻画个体文物风化程度、埋藏环境的差异；对于风化后测量值恰好为0的成分（例如高钾玻璃风化后K2O经常测到0），乘法缩放对0不起作用，无法反推出非零值——这是一个真实存在的模型局限，在后文如实说明。

**候选方案C（物理扩散模型，考虑后放弃）**：把风化过程建模为离子在多孔介质中的扩散/浸出过程（类似固体中的Fick扩散方程），需要时间、埋藏环境浓度梯度等参数。
- 数据里没有风化时长、埋藏环境浓度等信息，也没有同一文物随时间变化的序列数据（只有截面式的一次检测），此模型的参数根本无法从现有数据估计，属于"为了显得高级而堆砌复杂度"，不采用。这一决策也呼应了任务说明里"不要n板斧"的要求——机理复杂的模型如果数据支撑不了，就是过度建模。

**最终选择**：方案B（组均值比例反演），并用方案A留下的5个真配对样本中的2个（文物49、50，有完整"常规点+未风化点"配对，铅钡型）做诚实的留出验证。

### 2.2 问题2：分类规律 + 亚类划分

**分类规律部分**：
- 候选方案a：用全部14种成分做随机森林/SVM等黑箱分类器。
- 候选方案b（最终采用）：只用两个物理意义明确的变量——K2O（高钾玻璃的定义性成分）与 PbO+BaO（铅钡玻璃助熔剂的定义性成分）——做逻辑回归/浅层决策树。
- 理由：题目原文本身已经明确指出"高钾玻璃以草木灰(富钾)为助熔剂""铅钡玻璃以铅矿石(富铅钡)为助熔剂"，这是有明确化学机理支撑的先验知识，不是需要从数据里"挖掘"出来的黑箱规律。实际计算显示两特征逻辑回归在留一法交叉验证（LOOCV）下达到100%准确率（56个有效样本全部判对），决策树的规则简洁到可以写成一句话："PbO+BaO > 6.99% 判为铅钡玻璃，否则判为高钾玻璃"。在这种情况下用随机森林等黑箱模型没有任何精度收益，只会牺牲可解释性——机理已经很清楚，不需要用复杂模型去"发现"一个本来就已知的规律。这也是任务说明强调的"先想清楚机理再选模型"的具体体现。

**亚类划分部分**：
- 候选方案a：AHP层次分析法/熵权法对各成分打分排序。
  - 明确排除：AHP、熵权法是多准则决策(MCDM)中用于"给方案打分排序"的方法，本质是有序打分问题；而这里要解决的是"发现同一大类内部是否存在若干个自然的成分子群"，是一个无监督聚类问题，二者问题类型完全不匹配，AHP/熵权法在这里没有适用的目标函数（没有"更优/更劣"的排序概念）。属于典型的"n板斧"误用场景，明确不用。
- 候选方案b：高斯混合模型(GMM)。
  - 高钾型样本量仅16，铅钡型40，而GMM要为每个簇估计协方差矩阵，在14维（或选取的8-10维）特征空间下，16个样本远不足以稳定估计协方差，容易退化，故放弃。
- 候选方案c（最终采用）：对数变换（应对成分数据右偏、且含0值需要伪计数处理）+ 标准化 + K-Means聚类，用轮廓系数(silhouette score)在k=2,3,4之间选择最优k；聚类特征上，剔除已经在主分类里用到、且组内已经"没有信息"的定义性成分（如高钾组的K2O，因为组内该型全部文物K2O普遍偏高、区分度已经耗尽在"高钾/铅钡"这一步），保留能反映"同一大类内部配方微调"的主/次成分（SiO2、Na2O、CaO、MgO、Al2O3、Fe2O3、CuO、P2O5，铅钡型另外保留PbO、BaO因为这两者在铅钡型内部差异仍然很大）。
  - 理由：K-Means简单、可解释、对小样本更稳健；对数变换是处理成分数据（compositional data）的标准做法（避免了0值和右偏分布对欧氏距离的扭曲）；用轮廓系数客观选k，避免主观拍脑袋。

### 2.3 问题3：未知样本分类

直接复用问题2里验证过的两特征（K2O、PbO+BaO）逻辑回归/决策树规则，不再另起炉灶训练新模型——因为这本身就是分类规则的"部署"环节，重新训练一个更复杂的模型不会提升可靠性（问题2的LOOCV已经是100%），只会增加不必要的复杂度。

### 2.4 问题4：成分关联分析

- 候选方案a：偏相关网络 / Graphical Lasso 精度矩阵估计。
  - 高钾型样本量16，若要估计14维变量间的偏相关（协方差矩阵求逆），样本量远小于变量数的合理量级要求，估计出的偏相关矩阵会非常不稳定，弃用。
- 候选方案b（最终采用）：Pearson相关系数矩阵 + 显著性检验（|r|>0.3, p<0.05筛选），并用Fisher z变换检验"同一成分对在高钾组和铅钡组的相关系数是否有统计学显著差异"，这是比较两组相关结构差异的标准做法，计算量小、可解释、适合本题的小样本规模。

## 3. 关键假设及其论证

1. **假设1：85%~105%为有效数据（题目已给定，直接采纳）**。论证：这是题目原文明确写出的规则，不是我自己假设的，必须遵守。实际检查发现表单2的69个采样点中有2个（文物15、17，恰好都是"高钾-无风化"）成分和分别只有79.47%和71.89%，低于85%阈值，予以剔除。这两件文物因此在表单2中没有任何有效测量点，在问题2-4的建模中被整体排除（表单1原有58件，最终建模用56件）。

2. **假设2："常规采样点"代表该文物在该风化状态下的典型成分，"未风化点/严重风化点"是特殊标注的极值采样点，不代表文物整体**。论证：题目附件说明里明确写了"(3)未风化点是风化文物表面未风化区域内的点""(4)严重风化点取自风化层"，这些是文物表面异质性的局部采样，用来做前后对比、校准是合理的，但如果把它们和常规点混在一起算"该文物的代表成分"会引入系统偏差（比如把风化文物的"内部残留未风化小块"当成整体代表，会低估风化影响）。因此我在构建"文物级代表成分"时区分对待：无风化文物直接用常规点均值；风化文物如果有实测"未风化点"，直接用未风化点均值作为该文物的"原始成分"最佳估计（因为这是**实测的**残留原始成分，比任何反演模型都更可信）；风化文物若没有未风化点标注，才用问题1的组均值比例反演模型做校正。

3. **假设3：风化对成分比例的影响可以用"分型-分成分"的线性缩放系数近似**。论证：这是一个简化假设，其合理性用留出验证做了检验（见第4节）——用文物49、50两个真实同时有"常规点+未风化点"的铅钡型样本做验证，模型对主量成分（SiO2、PbO、CaO等）的预测误差在1~4个百分点量级，对微量成分（如BaO在文物50上误差达11.5个百分点）误差偏大。这说明线性缩放假设对主量成分基本适用，对微量成分不够可靠——这一点在论文里如实报告，不夸大模型能力。

4. **假设4：分类特征选K2O和PbO+BaO是"结构性假设"而非数据驱动挖掘**。论证：题目背景段落本身把"助熔剂种类"作为高钾玻璃和铅钡玻璃的定义依据，K2O、PbO、BaO是助熔剂残留的直接化学指标，这是有机理支撑的先验选择，不是在14个变量里"试出来"的。用t检验做了事后检验：PbO在两型间的差异 |t|=16.52, p=2.46×10⁻¹⁹，BaO |t|=8.22，K2O |t|=6.87，均远超其它成分（第二梯队SrO |t|=6.58，CaO |t|=6.34），支持了这个先验选择。

## 5. 遇到的问题与调整

1. **最初尝试的"个体级配对回归"方案在数据检查阶段就被放弃**：一开始设想用同一文物的"风化点 vs 未风化点"直接配对做回归，但实际枚举附件表单2的采样点标签后发现，绝大多数标注"未风化点"的文物根本没有配对的"常规点"测量（比如文物23、25、28、29等，表单2里只有一条"23未风化点"记录，没有"23"这条记录）。这迫使我改用组均值比例反演，把这些孤立的未风化点样本转而用作**问题2-4文物级代表成分表**中风化文物的"金标准"原始成分（比模型反演更可信），而不是浪费掉。

2. **pandas的astype(str)在某个版本下对混合float/str列出现了预期外行为**（`标记` 列部分为NaN(float)部分为str，`.astype(str)` 没有把所有NaN正确转成字符串"nan"，导致后续`in`操作报`TypeError: argument of type 'float' is not a container`）。改用 `.fillna('').astype(str)` 解决。这是一个环境/版本兼容性问题，记录以便日后遇到同类问题时可以快速定位。

3. **高钾玻璃亚类聚类第一版把PbO/BaO/SnO2也纳入聚类特征**，导致k=2的"最优"聚类被单个离群样本（18号，SnO2异常高）主导，聚类失去实际意义。调整为：先明确"聚类特征选择原则"（排除已用于主分类的定义性成分、排除过于稀疏的痕量成分），再重新聚类——虽然18号仍然是离群点（因为它本身CaO/Fe2O3/CuO确实为0，是真实的极端样本，不是被误选的特征造成的伪影），但至少排除了明显不该用的冗余特征干扰。

4. **环境没有预装scikit-learn/scipy**，用venv单独安装（`python3 -m venv venv2 && pip install scikit-learn scipy openpyxl pandas numpy`），避免污染系统Python环境，也规避了Homebrew Python的PEP668限制。

## 6. 自我批判：局限与改进方向

1. **风化反演模型（问题1核心模型）本质是"组均值比例缩放"，是本题里方法论上最薄弱的一环**。真实的化学风化过程涉及扩散、离子交换、pH、埋藏时长等因素，不是一个简单的线性比例关系；用留出验证（仅2个真实配对样本）也只能算初步的合理性检验，样本量太小，无法对模型误差做严格的统计推断。如果重来，会尝试：(a) 寻找是否有考古学文献报道过古代玻璃风化的时间序列/半定量风化程度指标，将风化程度作为连续协变量而非"风化/无风化"二元变量纳入模型；(b) 对无法用乘法缩放处理的"测量值为0"情形，设计更谨慎的处理方式（如非线性/加性修正、或者干脆对这类成分明确标注"反演不可靠"，不给出具体数值）。

2. **高钾玻璃亚类划分（n=16）样本量太小，聚类结果的可信度有限**，被单个离群样本主导，没有得到一个有丰富解释力的多亚类结构。如果重来，会考虑：(a) 明确把18号列为异常值单独讨论，不纳入聚类主流程；(b) 引入更多考古学背景变量（纹饰、颜色、出土地点等，如果数据可得）辅助验证聚类的合理性，而不是仅凭化学成分的无监督聚类。

3. **相关性分析（问题4）用的是Pearson线性相关，但成分数据存在"和为100%"的负相关偏置（compositional closure effect）**——任何两个成分只要其它成分不变，此消彼长会天然带来负相关，这是成分数据分析里的经典陷阱。更严谨的做法应使用中心对数比变换（CLR）后的相关系数，或用于成分数据的专门方法（如Aitchison几何）。本次分析为了保持结果直观、可读性强，直接用了原始百分比的Pearson相关，这是一个需要在论文里明确声明的方法论局限，不能把结论过度泛化为"因果关系"或"未经调整的独立关联"。

4. **问题2的分类特征选择虽然有机理支撑、且效果极好（LOOCV 100%)，但这本质上是"用答案（已知的类型标签）反过来验证了我们对机理的理解"，而不是纯粹的无监督发现**。如果题目要求的是"完全不依赖已知标签、从58个样本的14维成分中自动发现两个天然簇"，那么应该做一次不利用标签的纯无监督聚类（比如对全部56个有效样本一起做K-Means/层次聚类，k=2），检验其是否自动收敛到与已知标签一致的划分，以此作为"分类规律确实客观存在"的更强证据。这一步在时间预算内未做，是一个可以补充的验证实验。

5. **数据量偏小**（56个有效文物，8个未知样本），所有的统计检验（t检验、卡方检验、Fisher z检验）在小样本下的功效(power)有限，尤其是"纹饰/颜色与风化关系"这类多分类交叉表，很多格子的期望频数<5，卡方近似不完全可靠，论文里已如实标注"仅供参考"，但这类结果的可信度终究弱于类型-风化关系的结论。
