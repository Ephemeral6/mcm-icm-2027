## 1. 题目理解与拆解

题目给出一个"浮子+振子+PTO(弹簧+阻尼器)"两体波浪能装置。核心物理图像：

- **浮子**：圆柱壳体（下）+ 圆锥壳体（上），质量均匀分布在壳体表面，浸没在水中，受波浪激励力、
  附加惯性力、兴波阻尼力、静水恢复力（矩）。
- **振子**：穿在中轴上的实心圆柱体，密封在浮子内部，通过 PTO（弹簧+阻尼器）与浮子（经中轴底座）
  相连。
- **PTO 做功** = 阻尼器耗散的能量 = "可利用的电功率"。

四问的结构非常清晰，是一个循序渐进的建模任务：

1. **问题1**：只考虑垂荡（1自由度平动），给定阻尼形式（线性/幂律非线性），**正问题**——列出运动
   微分方程，代入参数数值仿真，输出时间序列。
2. **问题2**：仍是垂荡，但阻尼系数是待优化的设计变量，**反问题**——在约束域内找使"PTO平均输出
   功率"最大的阻尼参数。
3. **问题3**：垂荡+纵摇耦合（2个平动/转动自由度 × 浮子/振子 = 4个自由度），阻尼系数给定，仍是
   正问题，但多了转动子系统（扭转弹簧+旋转阻尼器）。
4. **问题4**：垂荡+纵摇耦合，两个阻尼系数（直线阻尼器、旋转阻尼器）都是待优化设计变量，反问题。

可以看出，问题1→2 是"平动子系统"的建模到优化，问题3→4 是把"平动子系统"和一个新增的"转动子系统"
放在一起，建模到优化。抓住这个结构后，代码复用就顺理成章：转动子系统在数学形式上和平动子系统是
完全同构的二阶ODE对，只是把 `位移z→转角θ`、`质量m→转动惯量I`、`力f→力矩L`、`线阻尼c→旋转阻尼c_rot`
做符号替换。

## 2. 候选建模思路对比

### 2.1 运动方程的建立方式：候选A（牛顿法/受力分析）vs 候选B（拉格朗日法）

- **候选A（受力分析，牛顿第二定律）**：对浮子、振子分别列出所有作用力（波浪激励力、附加惯性力
  [等效为附加质量乘以加速度，移到左边]、兴波阻尼力、静水恢复力、弹簧力、阻尼器力），直接写出
  两个耦合的二阶常微分方程。
- **候选B（拉格朗日法）**：写出系统总动能、总势能、耗散函数，用欧拉-拉格朗日方程推导。

**选择**：候选A。理由：题目本身用的语言就是"附加惯性力（矩）""兴波阻尼力（矩）""静水恢复力（矩）"
这种"以力/力矩为单位"的描述，直接对号入座写牛顿方程最省事、最不容易出符号错误；本题只有2个
自由度（每个子系统），没有复杂的约束坐标变换，拉格朗日法的优势（自动处理约束）在这里发挥不出来，
反而多一层"求偏导"的出错风险。所以候选A胜出。

### 2.2 非线性阻尼下最优参数的求解方式：候选A（直接时域数值积分+全局搜索）vs 候选B（频域复振幅法，
仅适用于线性）vs 候选C（等效线性化/谐波平衡法）

这是本题求解过程中**思路调整最大的一步**，详细过程见第5节"遇到的问题与调整"。这里先说结论：

- 问题2情况(1)、问题4（阻尼系数为常数，线性系统）：用**频域复振幅法**（候选B）。线性时不变系统在
  简谐激励下稳态响应必为同频简谐，可以直接解2×2复数线性方程组得到响应幅值，进而解析算出平均功率
  关于阻尼系数的函数，用一维有界优化即可精确求最优解，不需要数值仿真，又快又准。
- 问题2情况(2)（幂律非线性阻尼 c|v|^p sign(v)，p∈[0,1]）：最初尝试候选A（直接对非线性ODE做
  网格搜索+局部优化），**踩了一个数值陷阱**（详见第5节），后改用候选C——等效线性化（谐波平衡）法：
  假设稳态相对速度近似为主谐波 V·cos(ωt+φ)，用能量耗散等效的方法把非线性阻尼力换算成一个依赖于
  振幅 V 的等效线性阻尼系数 c_eq(V)，再用和候选B完全相同的频域方法自洽迭代求解 V，最后代回算平均
  功率。这个方法在 p=1 时能与频域精确解完全重合（数值验证误差 <1e-4），在 p=0.5 时与直接时域仿真
  相差约1-2%，精度足够、且计算成本从"分钟级卡死"降到"秒级完成"。

### 2.3 浮子的静水恢复刚度：候选A（假设水线始终落在圆柱段，恢复刚度为常数）vs 候选B（考虑浮子
沉没到圆锥段的非线性恢复力）

- **候选A**：线性波浪理论下小幅振荡，恢复刚度取平衡吃水处的水线面积对应值，视为常数
  k = ρg·A_waterline。
- **候选B**：若浮子垂荡幅度大到水线进入圆锥段，恢复力将随位移非线性变化（水线面积随浸没深度
  变化）。

**选择**：先用 Python 验证平衡吃水深度是否落在圆柱段内（见第4节代码），结果吃水 2.267 m < 圆柱段
高度 3 m，且从问题1、3的仿真结果看垂荡振幅在 0.3~0.9 m 量级，水线仍稳定处于圆柱段（3 m 高度有
充分裕量），候选A的线性化假设成立，采用候选A。这也是题目本身反复强调"线性周期微幅波""线性化"的
用意所在——不是我们偷懒，是题目设定的物理场景本来就该用线性理论。

### 2.4 问题3/4 转动子系统与平动子系统的关系：候选A（完全解耦，独立求解）vs 候选B（考虑质量/惯量
耦合项，联立6自由度或更高阶方程）

- **候选A**：认为垂荡运动（驱动直线阻尼器）与纵摇运动（驱动扭转弹簧+旋转阻尼器）在题目给定的
  线性化框架下彼此独立——浮子只受垂荡激励力矩驱动纵摇，垂荡激励力驱动垂荡，两个子系统的运动
  方程形式相同、参数不同、互不出现在对方的方程里。
- **候选B**：更严格的六自由度刚体动力学中，垂荡加速度与纵摇角加速度之间存在惯性耦合项（尤其是
  当振子偏离浮子质心时），需要联立更大的方程组。

**选择**：候选A。原因：(1) 题目附件3已经把"垂荡"和"纵摇"的附加质量/附加转动惯量/兴波阻尼系数/
激励力（矩）振幅**分列成互不干扰的两组参数**，这本身就是线性水动力学中典型的"垂荡-纵摇解耦"
处理方式（对于关于纵轴对称的浮体，在小幅振荡假设下垂荡与纵摇的水动力耦合项通常可以忽略或已经
隐含在附件给出的独立系数中）；(2) 题目文字也是分别描述"浮子只做垂荡运动"（问题1/2）和"浮子只做
垂荡和纵摇运动"（问题3/4），没有给出耦合项的数据（如垂荡-纵摇耦合附加质量），说明命题组的预期
解法就是把两个子系统当独立自由度处理。采用候选B需要的耦合惯量数据附件里也没有给，做不出来。
这是一个**受数据可得性约束的、明确声明的简化假设**，在论文"模型假设"与"模型评价"两节都会写清楚。

## 3. 关键假设的论证

1. **浮子壳体质量均匀分布在圆柱侧面+圆柱底盖+圆锥侧面（圆锥顶端无盖）**：题目原文"浮子由质量
   均匀分布的圆柱壳体和圆锥壳体组成"，且"忽略中轴、底座、隔层及PTO的质量"，因此壳体的质量应只
   分布在真正构成浮子外壳、需要密封浮子内部空间的曲面上——圆柱侧面、圆柱与圆锥连接处以下的
   平底盖（浮子最底端需要封闭）、圆锥侧面（圆锥顶点天然收拢，无需再加盖）。隔层是浮子内部的
   安装支撑面，题目明确说忽略其质量，故不计入壳体面积。
2. **系统平衡时水线落在圆柱段、水线面积为定值 πR²**：见2.3节论证，已用 Python 数值验证
   （吃水2.267m < 圆柱段3m）。
3. **纵摇转动惯量的计算轴选在浮子壳体自身质心处**：标准刚体摇荡（纵摇）方程 I·θ'' = M 中的转动
   惯量应取绕质心的转动惯量（题目附件给出的静水恢复力矩系数、附加转动惯量、兴波阻尼力矩系数均是
   针对浮子本身的纵摇自由度给出的，按照水动力学惯例，这些系数对应的参考点通常就是浮体重心/浮心
   附近），采用质心作为参考点是最标准且最省事的处理。
4. **振子绕转轴（隔层铰接点）的转动惯量近似取其自身绕质心的转动惯量，忽略偏置**（问题3/4）：
   这是本题**最大的不确定性来源**，题目及附件都没有给出振子在中轴上的初始安装位置到隔层铰接点
   的距离。我们用弹簧原长(0.5m)与初始压缩量(0.298m，由振子重力÷弹簧刚度算出)估算振子中心到
   铰接点的距离约为0.2m量级，属于"小偏置"，因此用平行轴定理修正量 m₂·d² 相对振子自身惯量
   （202.75 kg·m²）是同量级但不是主导项。我们在4.4节做了偏置量 0~0.5m 的敏感性扫描，
   结论对该假设不是特别敏感（θ1(40s) 从 -0.0333 rad 变到 -0.0424 rad，相对变化约27%，量级
   不变），把这个局限性写进"模型评价"里，不回避。
5. **忽略摩擦、忽略中轴/底座/隔层/PTO质量**：题目原文明确给出的假设，直接采用。
6. **浮子只做垂荡（问题1/2）或垂荡+纵摇（问题3/4）刚体运动，其余自由度（横摇、艏摇、纵荡、
   横荡）恒为零**：题目原文明确限定，直接采用。
7. **弹簧、阻尼器均视为理想元件（无质量、无迟滞）**：题目描述弹簧力正比于相对位移、阻尼力正比于
   相对速度（或速度幂函数），未提及非理想特性，按题目定义直接采用。

### 4.3 问题1：垂荡运动仿真（`heave_model.py`）

```
(m1+m_add)·z1'' = f·cos(ωt) - c_wave·v1 - k_heave·z1 - k_spring·(z1-z2) - F_pto(v1-v2)
m2·z2'' = k_spring·(z1-z2) + F_pto(v1-v2)
```

我们还做了求解器容差收敛性检验（rtol/atol 从1e-6加密到1e-10，t=100s处 z1、v1 结果小数点后
6位完全不变），确认仿真结果已经收敛，不是数值噪声。

### 4.6 问题2 情况(2)：非线性阻尼——一次真实的数值陷阱与调整

这是本次求解中**最值得如实记录的一段过程**。

**第一次尝试（失败）**：用 `scipy.integrate.solve_ivp`（自适应RK45）对每个候选 (c,p) 网格点直接
做时域仿真求平均功率，再用 Nelder-Mead 局部优化。运行后发现**单次调用耗时异常长**（6分钟仍未
跑完一个12×10的粗网格+精细优化），Bash 命令长时间无输出。

**诊断**：怀疑是 p<1 时 `|v|^p·sign(v)` 在 v=0 附近导数趋于无穷（p→0 时趋于干摩擦继电器特性），
自适应步长求解器在速度过零点附近反复缩小步长、大量拒绝迭代，导致极慢。

**调整1（仍然错误，但过程有价值）**：改用定步长 RK4（`fast_sim.py`），速度确实快了很多
（单次仿真0.05秷），网格搜索几秒内跑完。但optimize结果非常可疑：

```
粗网格(25x19=475点)最优: P=6025.922 W  c=100000.0  p=0.100
精细优化(40周期): c_opt=100000.00, p_opt=0.0952, P_max=6215.2027 W
```

P=6215W，是线性情况(229W)的**27倍**，直觉上不合理（同一个波浪激励下，仅仅换一种阻尼力的
速度依赖关系，理论上不可能让平均功率暴涨27倍——线性阻尼在谐振点附近本身已经是能量提取效率很高
的方案）。于是没有直接采信这个数字，做了收敛性检验：把步长从 T/300 加密到 T/2400（细4倍）：

```
dt=T/300:  P=6092.98 W
dt=T/600:  P=3176.81 W
dt=T/1200: P=1633.94 W
dt=T/2400: P=784.37 W
```

**功率估计随步长加密不断腰斩，完全没有收敛迹象**——这证实了之前的诊断：p≈0.095时阻尼力函数
接近阶跃/继电器特性，定步长RK4在速度过零点附近产生了虚假的高频数值噪声，"优化"出来的所谓
最优点其实是**数值伪影，不是真实物理解**。这是一个必须放弃的错误方向，如实记录，不能把
6215W这个数字写进论文。

**调整2（真正采用的方法）**：改用**等效线性化（谐波平衡）方法**（`describing_function.py`）：
假设稳态相对速度近似为单一谐波 V·cos(ωt+φ)，则一个周期内非线性阻尼耗散的能量
`E = c·V^(p+1)·I_p/ω`（I_p=∫₀^{2π}|cosθ|^{p+1}dθ 用数值积分算出），与"等效线性阻尼系数
c_eq(V)=c·V^{p-1}·I_p/π 下线性阻尼器耗散的能量"相等；再用与问题2情况(1)完全相同的频域复振幅
方法，把 c_eq(V) 代入线性系统求出新的响应幅值 V_new=ω|Z1-Z2|，自洽迭代到收敛。

一致性检验（真实输出）：

```
[检验1] p=1, c=37193.81: 等效线性化法 P=229.3339 W, 频域解析 P=229.3339 W  (Ip=3.1416, 理论应=pi)
[检验2a] p=0.5, c=10000: 等效线性化法 P=227.1058 W
[检验2b] p=0.5, c=10000: 精确时域仿真(solve_ivp,20周期,rtol=1e-6) P=229.8226 W
```

- 检验1：p=1时非线性阻尼退化为线性阻尼，等效线性化法与频域精确解**完全重合**（229.3339 W，
  6位有效数字一致），且中间量 I_p 精确等于理论值 π，说明代码实现正确。
- 检验2：p=0.5（问题1里已经算过的、不那么病态的非线性情况）时，等效线性化法（227.1W）与
  精确时域仿真（229.8W）相差约1.2%，在工程可接受范围内，说明"单一谐波近似"对这个系统是合理的
  近似（真实响应中高次谐波分量不大）。

在此基础上用等效线性化方法对 (c,p)∈[0,100000]×[0,1] 做完整网格搜索+局部优化：

```
网格(40x20=800点)最优: P=229.334 W at c=11153.8, p=0.500
精细优化: c_opt=11001.320 N.s/m^p, p_opt=0.4949, P_max=229.3339 W
最优点处相对速度幅值 V=0.1110 m/s

沿 p=1 (线性极限) 扫描 c 的功率曲线两端点核对：
  c->0: P=0.0352 W ; c=100000: P=7.6260 W
  p->0 (c固定于最优): P=0.0000 W ; p=1: P=125.1195 W
```

**一个重要的发现**：最优平均功率 **229.3339 W，与情况(1)线性阻尼的最优功率精确相等**（小数点后
4位都一样）！这不是巧合，而是等效线性化方法的必然结果：在"稳态响应只有单一谐波"的近似框架下，
任何非线性阻尼律对系统的影响都完全由标量等效阻尼系数 c_eq 决定，而线性阻尼恰恰是"用最小代价
（无需引入额外谐波失真）达到任意目标 c_eq"的阻尼律——所以线性阻尼已经是这一框架下的**帕累托最优
解**，幂律非线性阻尼在最好情况下也只能与之打平，不会更优。(c,p) 的最优解并不唯一，而是一条
"等效阻尼系数达到线性最优值"的曲线/山脊，我们给出的 c≈11001, p≈0.495 只是这条山脊上一个代表点
（用时域仿真复核，实际功率约233.5W，与理论预测229.3W接近，在等效线性化方法的近似误差范围内，
见4.7节完整复核）。这个结论本身对论文的"结果分析"是一个有价值的、诚实的洞见，而不是简单地报出
一个孤立的最优数字。

确认结论对振子转动惯量的不确定性是稳健的。

## 5. 遇到的问题与调整（汇总）

1. **数据文件误判**：附件目录中的 `result*.xlsx` 起初以为可能是标准答案，打开后发现只有表头无
   数据，是官方发放的填写模板，及时纠正认知，确认必须自己完整计算。
2. **numpy版本API变化**：`np.trapz` 在所用 numpy 2.5 环境中已被移除，改用 `np.trapezoid`，
   一次性小修复。
3. **非线性阻尼优化的数值陷阱**（最重要的一次调整，详见4.6节）：直接时域仿真+网格搜索在
   p→0（干摩擦极限）附近产生了不收敛的数值伪影（"最优功率"6215W实际上是虚假解），通过
   步长加密收敛性检验发现问题，改用等效线性化（谐波平衡）方法重新求解，得到更小、更可信、
   且有清晰理论解释（与线性阻尼情况的最优功率恰好相等）的答案。这是本次求解过程中最有教育
   意义的一步：**任何"优化出来的极端结果"都应该先做收敛性/合理性检验，再采信**。
4. **后台长时间无输出的排查**：Python 向管道/文件写 stdout 时默认有缓冲，导致命令"看起来卡住"
   实际是缓冲区未刷新；用 `ps aux` 检查真实进程状态、`python3 -u` 关闭缓冲，配合 Monitor
   工具等待进程结束，逐步定位到真正耗时的是"自适应ODE求解器在非光滑阻尼力函数处的步长塌缩"，
   而不是代码逻辑错误。

   把这一点列为"需要补充说明的建模假设"，并尝试用能量法（例如假设振子转动惯量与其相对浮子的
   静态平衡位置自洽求解）进一步收窄不确定性。
2. **垂荡-纵摇解耦假设未经严格的多自由度刚体动力学推导验证**，只是基于附件数据的分列方式和

   可忽略，但这需要题目未提供的附加水动力系数（如垂荡-纵摇耦合附加质量），已超出附件给定
