# 思考过程记录：CUMCM 2021 B题《乙醇偶合制备C4烯烃》

> 本文件是解题过程的留痕记录，如实记录推理轨迹、实际跑过的代码与真实输出、走过的弯路与最终取舍。执行者为 AI（Claude），非真人参赛队员。

## 一、题目理解与拆解

先读原题（`/Users/ephemeral/美赛/13-国赛CUMCM/真题2021/2021_B题.md`）。题目给的信息量不大，核心是：

- 反应体系：乙醇在 Co/SiO2（活性组分为负载在 SiO2 上的 Co）与 HAP（羟基磷灰石，一种碱性载体）组成的双功能催化剂上偶合，生成 C4 烯烃（目标产物）、乙烯、乙醛、C4-C12脂肪醇、以及其它副产物。
- 三个可调的"催化剂组合"变量：Co 负载量（wt%）、Co/SiO2 与 HAP 的装料质量比、乙醇进料浓度（ml/min）；再加一个工艺变量：温度。
- 两个关键指标：**乙醇转化率** = 100% × (进气量-剩余量)/进气量；**C4烯烃收率** = 转化率 × C4烯烃选择性（都是"%"意义下的乘积，即收率(%) = 转化率(%) × 选择性(%) / 100）。
- 附件1：21 组催化剂组合（A1-A14 装料方式I，B1-B7 装料方式II）× 各自若干温度点（250~450°C不等）的稳态性能数据。
- 附件2：固定某一催化剂组合、固定 350°C，考察**同一次实验、不同反应时间**（20~273 min）下转化率与各产物选择性的变化——这明显是在考察**催化剂的稳定性/失活行为**，和附件1的"多催化剂横向对比"是两个不同维度的问题。

题目四问的本质：
1. **描述性/拟合**：转化率-温度、选择性-温度的函数关系（附件1，逐个催化剂组合）；附件2 单次实验内的时间序列规律。
2. **归因/多元分析**：催化剂组合的三个因素 + 温度，对转化率、选择性分别有什么影响（方向、强弱、显著性）。
3. **优化**：在已有实验条件（离散网格）范围内，选哪个催化剂组合+温度使C4收率最大；再加一个约束 T<350°C 下的次优解；这里隐含了"能不能跳出已有网格去外推/插值找更优点"的建模空间。
4. **实验设计**：只给5次新实验的预算，属于经典的"序贯实验设计/响应面主动学习"问题——不能瞎猜，要基于前面模型给出的关键因子和不确定性来设计。

一开始我的第一反应是：这题看起来像"回归+优化+实验设计"三段套路，容易滑向"随便上个机器学习模型调个参数"的空对空套路。国赛评委明确反感这种不问机理、直接上黑箱模型的做法，所以我先花时间理解**反应机理**，再决定用什么模型。

### 反应机理的背景知识（我作为先验带入建模，不是拍脑袋）

乙醇偶合制 C4 烯烃的公认机理是"Guerbet 类" 串联反应路径：
1. 乙醇在 Co 活性位上脱氢 → 乙醛（需要金属/氧化还原活性位）；
2. 乙醛在 HAP 提供的碱性位上发生**羟醛缩合（aldol condensation）** → 巴豆醛/丁烯醛等 C4 中间体；
3. 中间体加氢、脱水 → C4 烯烃。

这意味着：Co 是"脱氢"功能的载体，HAP 是"缩合成链"功能的载体，两者必须**协同**（双功能催化剂），单独任何一个都做不出C4烯烃。这个先验后面被数据直接验证（见下文 A11 vs A12 的对照），也是我选择建模变量和写因果解释时的重要支撑，而不是纯粹回归系数堆砌。

## 二、候选建模思路的比较

### 问题(1)：转化率/选择性 vs 温度

- **候选A：Arrhenius(阿伦尼乌斯)动力学严格建模**，即用真实反应速率方程 r=k0·exp(-Ea/RT)，需要知道反应级数、进料浓度等，再反解转化率。
  - 问题：附件1只给了"稳态出口转化率"，没有给反应器停留时间分布、动力学级数等信息，无法严格标定Ea，会陷入"为了看起来专业硬凑动力学方程"的陷阱。
  - 决定：**放弃**严格阿伦尼乌斯定量标定，但保留"用指数模型去拟合转化率-温度关系、并与线性模型比较拟合优度"的做法，作为对Arrhenius类现象的定性佐证（活化能量级的现象学近似），不出现虚构的Ea数值。
- **候选B：逐催化剂组合的经验拟合（线性/指数/多项式）+ Pearson相关系数**，只做描述统计和拟合优度比较，不装作在做严格动力学。
  - 优点：诚实、可执行、能覆盖全部21组数据、能量化"温度效应有多强、多显著"。
  - **采用**。

### 问题(2)：催化剂组合因素 + 温度 对转化率/选择性的影响

- **候选A：随手上 AHP/熵权法**——不合适，AHP是主观权重决策法，这里是有真实观测数据的定量归因问题，用AHP纯属瞎胡闹，评委看了会直接扣分（也是题目附带原则里明确警告的"不要n板斧"）。
- **候选B：随手上 BP神经网络/深度模型**——21个催化剂组合、114条观测，样本量太小，神经网络会严重过拟合且不可解释，纯属"看起来高级实则空转"。
- **候选C：多元线性回归（含标准化系数）+ Bootstrap置信区间 + 偏相关分析**，辅以随机森林重要性和分组交叉验证做**稳健性交叉检验**（而非主模型）。
  - 需要先把"催化剂组合"的文本描述解析成结构化数值特征：Co负载量(wt%)、Co/SiO2与HAP装料质量比、乙醇流速(ml/min)、催化剂总质量(mg)。
  - 关键的额外思考：乙醇流速和催化剂总质量单独看都不是"纯"变量——真正决定停留时间(接触时间)的是两者的比值 W/F（催化剂质量/流速，类似空速的倒数）。如果分别把"总质量"和"流速"作为独立回归变量，会与"催化剂组合本身设计的初衷"不一致（题目里同一系列常常是固定总质量改流速，或固定流速改总质量），所以我构造了 `contact_time = 总质量(mg)/流速(ml/min)` 这个物理上更有意义的合成变量代替这两个原始变量，回归的可解释性显著提升（详见下文代码与R²对比）。
  - **采用**：以线性回归为主模型（可解释、可给显著性），随机森林仅作稳健性/非线性交叉验证的旁证，不作为最终结论的唯一依据。

### 问题(3)：优化催化剂组合+温度使C4收率最大

- **候选A：直接在114个观测点里取最大值（网格搜索/枚举）**——最诚实但只能给出"已测过的条件里最好的是哪个"，无法回答"有没有更好的没测过的条件"。
- **候选B：拟合响应面（多项式回归或随机森林），在观测变量的凸包范围内做插值搜索**，可以在未测过的网格点上给出预测最优区域，但**不能外推到凸包之外**（否则是编造）。
- 决定：**两者都做**。先用候选A给出"证据确凿"的经验最优解（附带T<350的约束版本），再用候选B的响应面模型做插值层面的探索，同时明确标注"这是插值预测，不是新的实验事实"，为问题(4)的实验设计提供依据。

### 问题(4)：增补5次实验的设计

- **候选A：全因子设计/正交表**——5次实验预算太少，摊不开一个像样的正交表，而且没有利用已有98%~99%置信的模型结论（温度和接触时间是主导因子，Co负载量对选择性有显著负效应，装料比对选择性有正效应）。
- **候选B：贝叶斯优化/响应面法的"序贯实验设计"（steepest ascent + 局部探索）**：以问题(3)找到的经验最优点（A3组合，400°C）为中心，沿着模型识别出的关键梯度方向（温度精细定位 + 接触时间进一步增大 + 装料比在大质量尺度下的探索）设计5个局部加密实验，并保留一个复现性实验应对附件2揭示的"催化剂会随时间失活"这一测量噪声来源。
  - **采用候选B**，因为它直接利用了前面模型的定量结论，而不是另起炉灶。

## 三、关键假设及其论证

1. **稳态测量假设**：附件1中每个"催化剂组合-温度"对应的转化率/选择性数值，视为该条件下的稳态（或近似稳态、短时间内的代表性）测量值，可跨催化剂组合直接比较。
   - 论证：附件2显示同一催化剂在350°C下随时间推移转化率从43.5%降到约29.9%（273分钟内降31.3%），说明"时间"确实是一个会显著改变转化率的变量。但附件1没有报告每个数据点对应的测试时刻，只能假设其为"标准化流程下、相近测试时段"采集，是题目设计上默认的可比性假设，我在论文里会明确写出这是一个限制，而不是回避。
2. **C4收率的乘积定义直接采用题目给定公式**：收率(%) = 转化率(%) × 选择性(%) / 100，这是题目原文明确给出的，不需要额外假设。
3. **接触时间替代变量 W/F 合理性假设**：用"催化剂总质量/乙醇流速"近似代表停留时间（虽然真实停留时间还与反应器几何、气体膨胀等有关，但在固定床积分反应器中 W/F 是最常用的空速代理变量），这是化学反应工程里的标准做法，不是我编造的。
4. **催化剂组合的三个连续特征（Co负载量、装料比、接触时间）在观测范围内近似连续可插值**：用于响应面方法，但不外推到观测范围以外，这个假设只在问题(3)(4)的"插值搜索"部分使用，且做了范围限制。
5. **A11（无HAP，只有惰性石英砂）可作为"缺少HAP"的对照组**：这不是我加的假设，是数据本身自带的一个准对照实验，我只是识别并利用了它。

**物理解释**（不是编的，是结合机理先验对回归系数的解读）：Co负载量越高，脱氢能力越强、乙醛生成越快，但如果HAP提供的缩合活性位跟不上，过量乙醛来不及缩合就会走副反应（数据里能看到高Co负载量组合A6、A10的乙醛选择性异常高），导致C4选择性反而下降——这与回归给出的"Co_wt对选择性负显著"完全吻合。装料比越高（Co/SiO2相对HAP越多）在小质量尺度上小幅提升选择性，可能是因为总碱性位仍然充足时增加脱氢活性位有利，但这一效应幅度不大（标准化系数仅0.12）。

**一个数据自带的"准对照实验"**：A11（50mg 1wt%Co/SiO2 + 90mg**惰性石英砂**代替HAP）与A12（50mg 1wt%Co/SiO2 + 50mg**HAP**，其余条件相同）几乎是理想的对照组。真实数据对比（350°C）：
```
A11: 转化率8.21%  C4选择性4.35%  收率0.36%
A12: 转化率19.91% C4选择性22.26% 收率4.43%
```
去掉HAP后C4选择性从22.26%暴跌到4.35%（降幅超过80%），转化率也大幅下降，这是对"HAP的碱性位是C4偶合缩合步骤的必需条件"这一机理假设的**直接数据证据**，比单纯回归系数更有说服力，我在论文里会重点引用这组对照。

### 4.5 问题(3)：优化

## 五、遇到的问题与调整

1. **附件2表头解析bug**：第一次用 `header=None` 后 `iloc[2:]` 切错了行，把子表头当成数据，导致 `astype(float)` 报错。通过打印完整原始表定位到应该是 `iloc[3:]`，修正后重跑通过。
2. **回归变量设计的返工**：最初直接把"总质量"和"乙醇流速"作为两个独立回归变量，R²偏低（转化率0.694、选择性0.530）且"装料比"完全不显著（p=0.766）。意识到这两个变量本质上共同决定"接触时间/空速"这一个物理量后，合并为 `contact_time = 总质量/流速` 重新回归，虽然R²略有下降（0.667/0.542，因为损失了一个自由参数），但系数的物理意义更清晰、标准化系数排序更符合Bootstrap稳健性检验，最终采用这个版本作为正式结论，而不是唯R²论。
3. **随机森林网格搜索给出的"最优预测点"收率反而低于实测最优点**：一开始以为响应面模型有bug，检查后发现是随机森学（max_depth=4限制+分段常数预测）本身的平滑效应导致，不是代码错误。调整了论文表述：只把响应面结果当作"验证主导因子方向"的旁证，不用它替代实测枚举的最优点结论，避免"模型比数据还权威"的本末倒置。
4. **A5、A6转化率在275°C出现非单调**：没有强行用模型消除这个"异常"，而是如实报告为局部实验波动，不纳入"温度单调提升转化率"这一整体结论的反例分量（21组里19组严格单调）。
