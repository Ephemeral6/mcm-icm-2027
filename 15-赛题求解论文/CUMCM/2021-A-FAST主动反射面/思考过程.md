## 1. 题目理解：这道题到底在问什么

先通读题目原文（`/Users/ephemeral/美赛/13-国赛CUMCM/真题2021/2021_A题.md`），拆解出以下要点：

1. **物理系统**：FAST 主动反射面由 2226 个主索节点、6525 根主索、4300 块反射面板、2226 套"促动器+下拉索"组成。基准态（不工作时）所有主索节点都精确位于一个半径 R=300.4m、球心在原点 C 的基准球面上。
2. **工作态**：观测某天体 S（方位角 α、仰角 β）时，需要把基准球面中一部分（口径 300m 范围内）"掰"成一个近似旋转抛物面，抛物面的对称轴是直线 SC，焦点落在与基准球面同心、半径差 F=0.466R 的"焦面"球上，具体落点是直线 SC 与焦面球的交点 P。
3. **执行机构**：每个主索节点连一根**定长**下拉索，下拉索下端接促动器顶端，促动器沿**基准球面径向**安装，顶端伸缩范围 **[-0.6, +0.6] m**（正方向为指向球心）。也就是说，节点的可控运动**只能沿它自己在基准态下的径向direction**发生，这是一个很强的机械约束，不是"节点可以随便移到抛物面上任意最近点"。
4. **三问结构**：
   - 问题1：S 在正上方（α=0°, β=90°，轴对称特例）——**确定理想抛物面**（不要求给出全部节点调节方案）。
   - 问题2：S 在给定斜方向（α=36.795°, β=78.169°）——**先定理想抛物面，再给出让实际主索网"贴近"该抛物面的调节方案**（每个节点新坐标、每个促动器伸缩量），要求按附件4格式输出。
   - 问题3：基于问题2的调节结果，用几何光学计算"接收比"（馈源舱 1m 直径圆盘接收到的信号 / 300m 口径内反射面总反射信号），并与不作调节的基准球面比较。

**这道题真正的建模核心不是"怎么算抛物面公式"（那是解析几何），而是两层递进的最优化问题**：
- 第一层（问题1、2的"确定理想抛物面"）：抛物面的**轴方向由 S 决定，焦点由焦面球决定，都是给定的**，唯一自由的是"抛物面沿轴向的位置"（顶点在轴上的深浅）——这是一个**单参数**优化问题：选这个参数使抛物面在 300m 口径内"最贴近"基准球面（从而让每个节点需要移动的量最小，保证促动器 ±0.6m 冲程够用）。
- 第二层（问题2的"调节模型"）：由于促动器只能让节点沿**自身基准径向**运动，"贴近"不能理解成节点移到抛物面上最近的欧氏距离点，而必须是"节点所在的径向射线与抛物面的交点"——这是本题里最容易被误解、也是我认为最重要的建模细节。

问题3 则是一个几何光学 / 蒙特卡洛光线追迹问题：把面板当作局部平面镜，平行光反射，看落点是否落在 1m 圆盘里。

## 2. 候选方案：我考虑过的建模思路

### 候选方案 A（最终采用）：焦点/轴固定的单参数抛物面族 + 节点径向投影 + 最小二乘拟合

利用题目已经把焦点和轴都钦定死的事实，把"确定理想抛物面"降维成一个**只有一个自由度 d（顶点沿轴的下沉深度）**的优化问题；节点的"贴近"用它自身径向线与抛物面的交点来定义（quadratic 求根），促动器伸缩量就是该交点到原节点的径向位移取负号。全局最优 d 通过在 d∈[-1,1] 上做稠密网格搜索（步长 5×10⁻⁴ m）来找，因为目标函数（各节点径向位移平方和）关于 d 是光滑的，数值上验证是单峰的，网格搜索足够稳、也足够精确，没有必要上更复杂的非线性规划求解器。

**为什么选它**：
- 完全尊重题目给的机械约束（节点只能沿基准径向运动）——这是物理上唯一"可执行"的调节方式，不是数学上任意的最近点投影。
- 自由度数与题目实际自由度（1个）完全匹配，没有引入题目不允许的自由度。
- 计算量小、全局最优有保证、结果可解释（d 就是"顶点相对基准球面下沉了多少"）。

### 候选方案 B（考虑后放弃）：一般 6 自由度抛物面最小二乘拟合（任意顶点、任意轴向、任意焦距，用 Levenberg-Marquardt 对所有口径内节点做"点到曲面欧氏最短距离"拟合）

这是最"标准"的曲面拟合思路：给定一堆散点，拟合一个二次曲面。但仔细想后放弃了，原因：
1. **物理上不可执行**：这种拟合允许节点朝任意方向移动到曲面最近点，但促动器只能沿径向运动，得到的"最优"曲面即使数学上拟合误差更小，也无法通过真实机构实现。
2. **违反题目给定的硬约束**：轴方向必须是直线 SC（否则馈源舱移动轨迹、焦面定义全部对不上），焦点必须在焦面球上直线 SC 与其交点处（否则馈源舱没法把接收面中心放在焦点上）。这两个都不是可以自由拟合的参数，而是题目直接给定的。放开这两个自由度求出的"更优"抛物面在工程上是没有意义的伪解。

所以候选方案B虽然是很多人第一反应会写的"标准套路"（类似不假思索套 AHP/熵权法的思路），但仔细读题后判断它不符合本题的物理约束，故弃用。这也是任务要求里强调的"不要n板斧"的一个具体体现——先想清楚机理，再选模型。

### 候选方案 C（用作敏感性对照，非主用）：极小化最大偏差（Minimax/Chebyshev）而非最小二乘

即选 d 使 max|Δᵢ| 最小，而不是 Σ Δᵢ² 最小。理由是促动器约束本质上是一个"硬上限"（±0.6m），minimax 直接对准这个约束。但后来发现：
- 最小二乘解在 300m 口径下已经远离 0.6m 上限（问题2里实测 max|Δ|=0.4506m，只用了 75% 的冲程），说明约束根本没有起作用（非紧约束），此时 LS 与 minimax 的差别只是"锦上添花"，而 LS 额外还能让**整体**偏差更小（这对问题3的聚焦质量更有意义，minimax 只管最差的一个节点）。
- 因此把 minimax 只作为敏感性分析的一个对照维度保留（见第5节），主模型仍用最小二乘。

## 3. 关键假设的论证

1. **节点调节只能沿其基准态径向方向运动**。
   论证：题目原文"促动器沿基准球面径向安装……径向伸缩"，且我用真实数据验证了：促动器地锚点(下端点)—促动器顶端(基准态上端点)—对应主索节点三点，在基准态下**近似严格共线**（对全部 2226 个促动器计算方向余弦，`cos ≈ 0.9999998~1.0`，见第4节代码输出）。这说明"节点-下拉索-促动器"整个链条在设计上就是沿径向布置的，调节时只是让这条径向线上的点沿线滑动，因此"节点位移=促动器伸缩量（正负号相反）"这个简化是有真实数据支撑的，不是凭空假设。
   （这条假设同时也解释了为什么"下拉索长度固定"不等于"下拉索长度处处相等"——数据显示下拉索长度从 1.23m 到 49.45m 不等，因位置而异，但**每根**下拉索长度对该节点是固定值，这与"节点沿径向滑动、促动器与下拉索整体刚性沿径向串联移动"的图像完全自洽。）

2. **基准球面半径 R=300.4m 及焦径比 F/R=0.466 直接取自附件与题目**。
   论证：对附件1全部 2226 个节点坐标计算 |X|，得到 300.3993~300.4007m（均值 300.40001m），与题目"半径约300米"及通常引用的 R=300.4m 完全吻合，故直接采用 R=300.4（而非"约300"这种模糊值）。

3. **300m 口径的节点筛选标准：节点基准位置在垂直于观测轴平面上的投影半径 ρ≤150m**。
   论证：题目定义"300米口径"是相对于观测方向（对称轴）的口径，用节点到轴线的垂直距离（而不是到 C 的三维距离）衡量是标准的光学口径定义；这与"接收比"定义里"300米口径内反射面的反射信号"也是同一个筛选集合，逻辑自洽。

4. **邻近节点间距变化≤0.07%是一条软约束，不纳入抛物面拟合的目标函数，而是作为事后检验项**。
   论证：这是主索网的**结构完整性/防止面板被过度挤压**约束，属于"调节方案是否可实现"层面的检验，与"确定理想抛物面（让节点整体最贴近）"这个几何优化问题是两个不同层次的问题；把它硬塞进目标函数会让问题从"1维搜索"变成2226维带约束的二次规划，复杂度陡增。我采用"先求理想抛物面+径向调节，再检验该约束满足到什么程度"的分层策略，并在第4节如实报告了检验结果（约1/3的口径内边超过容差）——这是本方案一个诚实暴露的局限，见第6节自我批判。

5. **问题3的接收比用几何光学（面板视为局部平面镜，入射为来自 S 的平行光束）计算，馈源舱接收平面近似为过焦点、垂直于观测轴的平面圆盘**。
   论证：题目本身明确"电磁波信号及反射信号均视为直线传播"（附录假设4），平行光假设对应"天体在无穷远处"（射电天文标准假设）；馈源舱有效区域直径仅1米，相对于300米级别的尺度和140米级焦距是极小量，用"过焦点、法向为观测轴"的平面局部近似球面焦面上的圆盘，误差可忽略。

### 4.4 问题3：接收比的几何光学仿真（`part3.py`）

思路：把问题2调节后（口径内三点均在口径内的）1295块面板当作局部平面镜，来自 S 方向的平行光反射后，与"过焦点、法向为观测轴 n̂"的平面求交，判断落点是否落在半径0.5m的圆盘内；每块面板按投影面积加权（能量流守恒的一阶近似）。对基准球面（未调节）做同样计算作对比。

诊断2——排除"面板离散化(平面近似曲面)"这个因素，直接用抛物面在面板形心处的**解析法向量**（而不是三节点撑起的平面法向量）来反射：

→ 几乎完美聚焦到焦点（平均4.3cm，远小于0.5m接收盘半径）！这证明：**理想抛物面本身的聚焦性质是对的，模型和优化过程没有错误**；接收比只有百分之一量级的**根本原因是面板本身太大**——面板是刚性平面（弦近似曲面），单块面板边长约11.4米，其反射光斑本身尺寸就与面板尺寸同量级（约7~11米），远大于1米接收盘，因此即使把整块面板的"中心"对得再准，光斑本身"糊"在盘外的部分也无法收进1米盘——这是**面板离散化本身带来的物理极限**，不是调节精度不够的问题。

这是我认为本次求解过程中最有价值的一次"自我纠错/深挖"：一开始怀疑是bug，通过设计对照实验排除了模型逻辑错误，定位到真实的物理限制来源，并把这个发现写进了论文的结果分析部分。

## 6. 自我批判：局限与如果重来会怎么改进

1. **邻近节点间距≤0.07%约束未被纳入优化目标，仅事后检验**，结果显示约32%的口径内边、75%的边界边超出容差。这是本方案最大的诚实缺口。更完整的做法应该是把"确定理想抛物面（1个参数）"和"节点调节（2226个自由度，但每个自由度被约束在自身径向线上，所以实际上仍然是2226个标量自由度）"合并成一个**约束优化问题**：在满足所有边长变化≤0.07%（或至少≤某个更宽松但可分析的容差）的前提下，最小化节点径向位移的加权平方和，用二次规划(QP)或序列线性规划求解。这个改进方向是明确的，只是超出了本次示范解答的时间预算。
2. **边界处理粗糙**：口径边缘的节点"一侧动、一侧不动"是边长超差最严重的地方（74.9%超容差）。更好的做法是引入一个从口径内到口径外的**过渡/羽化区**（例如口径外一圈节点也做微小的、随距离衰减的调节），平滑过渡而不是硬边界一刀切。
3. **接收比仿真的面板反射模型是"整块刚性平面镜"**，没有考虑：(a) 面板拼接处的微小缝隙和公差；(b) 蒙特卡洛采样量对结果的估计误差（不同随机种子给出1.08%~1.30%的波动，说明还需要更大样本量或解析积分才能给出更精确的单一数字，我在论文里如实以区间/均值±波动范围呈现，没有假装是精确值）。
4. **参数寻优仅在给定口径半径150m、焦径比0.466的"标称设计"下做了1维网格搜索**，没有反过来做"给定促动器冲程，联合优化口径半径与焦径比"这种更高层次的系统设计问题——但这超出了本题问题1/2/3的要求范围，属于合理的范围裁剪。
5. 如果重来，我会优先把第1点（约束优化代替无约束最小二乘）做扎实，因为这是官方赛题真正强调的"反射面板调节约束"里最容易被简化掉、也最容易被评委抓到的一点。
