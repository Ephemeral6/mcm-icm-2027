## 一、题目理解：拆解这道题到底在问什么

先通读题目原文（`/Users/ephemeral/美赛/13-国赛CUMCM/真题2024/2024_D题.md`，并核对了 PDF 原件的图1、图2，因为文本抽取会丢失图片信息，而图1的坐标系方向、图2的阵列间距标注对建模至关重要）。

拆解出的关键信息：

1. **坐标系**：原点 O 是潜艇"定位值"在海面的投影；X 东、Y 南、Z 向下（深度）。潜艇航向方位角 β 是"正北顺时针转到航向"的角，图1里画得很清楚，是标准罗盘方位角。
2. **随机性来源只有一个**：潜艇真实中心位置 (X,Y,Z) 相对定位值 O 存在随机误差——X,Y 独立同正态 N(0,σ²)，Z 是单边截尾正态（不能比某个物理下限 l 更浅）。**题目没有描述深弹投放本身的散布误差**，所以深弹落点 (x0,y0) 应被当作攻击方可以精确控制的决策变量，而不是另一个随机变量。这是我读题时首先要确认的一点，因为如果把它也建模成随机变量，问题会变成两个独立随机源的卷积，题目没有给这个随机源的分布参数，说明它不是本题要建模的对象。
3. **命中的三种情形本质是一个分段布尔逻辑**：
   - (1) 落点在艇的水平投影内 **且** 定深引信设定深度 d 不浅于艇顶面深度 → 触发引信必然先起爆 → 100% 命中；
   - (2) 落点在水平投影内 **但** d 比艇顶面更浅（引信会在碰到艇体之前，即先到达设定深度）→ 提前由定深引信起爆，命中与否要看爆点是否在杀伤半径 R 内；
   - (3) 落点在水平投影外 → 深弹垂直下沉的路径根本不会碰到艇体，只能靠定深引信在深度 d 起爆，同样要看爆点到艇体的距离是否 ≤ R。

   这三条初看是三条独立规则，但仔细看会发现情形(2)(3)的判据其实是同一个"起爆点到艇体（长方体）的欧氏距离 ≤ R"，区别只在于情形(1)的条件是否满足。这提示我可以把它们布尔化简为一个更简洁的表达式（见下节"关键假设与推导"）。

4. **潜艇简化为长方体**，长100/宽20/高25（问题1、2给定数值），杀伤半径是"球形杀伤区"（题目原话"深弹在海水中的最大杀伤距离"，隐含杀伤概率在半径内外是阶跃的 0/1，不是随距离衰减的连续函数——这是题目自身表述带来的简化，不是我强加的）。

5. 三问的递进关系：问题1（Z无误差，只有水平误差）→ 问题2（Z也有误差，且是截尾正态，多了"决定引信深度"这个真正的优化问题）→ 问题3（多枚深弹阵列，"至少命中一枚"）。这是一个典型的"先单变量后多变量、先确定性后随机性"的递进结构。

## 二、候选建模方案对比

### 方案 A（被放弃）：把潜艇简化为质点，只用"杀伤半径"一个判据

最省事的做法是无视题目给出的长宽高，直接把潜艇当成一个点，命中判据统一为"起爆点到潜艇中心距离 ≤ R"。

**放弃理由**：题目明确给出了长方体三个尺寸，并且用整整一段文字区分了"触发引信/定深引信"两种起爆机制，如果不区分，这些数据和文字描述就完全没有用武之地。国赛出题人不会平白给出用不到的参数，这是一个强烈的信号：本题的核心难点就是要正确处理"体积目标+双引信"的几何关系，质点化是在回避题目真正要考察的内容。

### 方案 B（初版尝试，后来发现有问题）：只用"情形(1)"作为命中判据，把(2)(3)当作命中概率的"小修正"忽略掉

这是我最初的直觉：既然情形(1)是"只要落点在投影内、且引信设深合适，就 100% 命中"，很多网上能查到的思路都是直接把这一项当作命中概率的主体，(2)(3) 只是"打偏了但运气好蹭到杀伤半径"的小概率事件，可以估计一下量级、如果很小就不必展开写。

**发现问题的过程**：我先做了一个简单的假设——R=20m 相对潜艇半宽只有10m来说，"蹭到杀伤半径"的修正应该是小量级。为了验证这个直觉，我写了 `p1b_correction.py` 做了一次蒙特卡洛（详见下节"计算过程"），结果发现：

- 情形(1)对应的核心概率 P(A) = 0.0215（2.15%）
- 情形(2)(3)合起来贡献的"补充命中概率" = 0.0623（6.23%）

**补充项是核心项的近 3 倍！** 这完全推翻了我"这是小修正"的直觉。仔细想为什么：R=20m 这个杀伤半径，跟潜艇的**半宽**（10m）同量级、甚至是半宽的2倍——也就是说沿着潜艇宽度方向，只要落点偏出艇体边缘不超过20米（相当于总有效宽度从20米变成60米），照样能炸沉。这在物理上是完全合理的（杀伤半径本来就是用来覆盖"差一点点没直接砸中"的情况的），只是我一开始凭直觉低估了它的量级。

**调整**：放弃"只算情形(1)"的简化，采用完整的三情形模型，并且专门做了一次解析分解（核心矩形 + 上下边带 + 左右边带 + 四角，见下节）来把这个修正项的来源讲清楚，而不是笼统地说"蒙特卡洛算出来是这个数"。这是本题最重要的一次思路调整，也是这份解答区别于"套公式"式做法的地方。

### 方案 C（最终采用）：三情形布尔化简 + 解析式与蒙特卡洛交叉验证

设：
- A = {落点在水平投影内} ∧ {d ≥ 艇顶面深度}
- B = {起爆点到艇体（长方体）的欧氏距离 ≤ R}

可以证明：命中事件 = A ∨ B（见下节推导）。这个化简把三条文字规则统一成一个两项析取式，既严谨又便于后续做解析和数值计算。问题1、2、3 都基于这个统一表达式展开，只是随机变量的维度和决策变量的自由度不同。

## 三、关键假设的论证

写入论文正文的"模型假设"，每条给出为什么合理：

1. **艇体刚性长方体近似**：题目原文直接指定"将潜艇主体部分简化为长方体"，不是我加的假设，是题目条件。
2. **深弹严格垂直匀速下沉，水平落点在下沉过程中不变**：题目原文"深弹在水中垂直下降"，这是题目给的物理设定，我只是把它翻译成"下沉轨迹是一条垂直线，水平投影不随深度变化"这句可用于建模的话。
3. **触发引信接触即100%命中，无失效概率**：题目用"视为命中潜艇"这个措辞（而不是"以概率p命中"），说明本题不考虑引信可靠性，我们直接按题目定义的三个"命中"条件来算，不外插空气阻力、引信故障率等题目未给出的因素。
4. **杀伤半径内外是阶跃的0/1杀伤**：题目原话是"深弹在海水中的**最大**杀伤距离称为杀伤半径"——"最大杀伤距离"这个措辞本身就是阶跃语义（超过这个距离就不算杀伤范围了），不是连续衰减的杀伤概率密度函数。如果要建立连续衰减模型需要额外的物理参数（如冲击波衰减规律），题目没有给，属于过度建模，故采用题目字面语义的阶跃模型。
5. **潜艇水平定位误差各向同性**（X、Y 同方差 σ² 且独立）：题目直接给出。这条假设在数学上有一个很好的推论：X,Y 的联合分布是旋转不变的各向同性高斯分布，这让我可以把"艇体绕 β 角旋转"这个几何复杂度，通过坐标旋转变换完全"吸收掉"（详见推导），不需要对每个 β 单独算一遍。
6. **深弹落点 (x0,y0) 无投放误差，是确定性决策变量**：题目只描述了潜艇定位误差的分布参数，没有给深弹投放散布误差的任何参数，说明本题的随机性来源只设计为潜艇位置一个自由度。这是"读题读全"而不是"想当然加设定"的结果。
7. **问题3中9枚深弹落点阵列相对刚性、共用同一次潜艇位置实现**：飞机一次投掷9枚深弹，飞行时间差远小于潜艇静默状态下的移动能力（题目也说潜艇发现被跟踪后"关闭发动机、静默隐蔽"，即不再主动机动），所以合理地认为这9枚弹面对的是**同一个**随机实现的 (X,Y,Z)，而不是9次独立同分布的抽样。这一点在写蒙特卡洛代码时必须小心处理——不能对每枚弹单独抽样，而要对同一批 (X,Y,Z) 样本判断9枚弹是否"至少一枚命中"，这是问题3和"重复9次问题2"的本质区别（重复9次独立抽样会显著低估相关命中率的提升，因为它错误地假设了潜艇位置在9次投弹之间独立变化）。
8. **忽略多枚深弹之间的相互干扰**（如水中冲击波互相影响、艇体被击中后杀伤半径变化等）：题目没有给出这类二阶效应的参数，按题目字面条件处理。

## 四、模型建立与推导

### 4.1 坐标旋转 —— 把"任意航向角 β"问题降维成"轴对齐"问题

局部坐标 (u,v)：u 沿潜艇航向、v 垂直航向。由罗盘方位角的定义（正北顺时针转 β 到航向），航向单位向量在 (X东,Y南) 坐标系下是 (sinβ, -cosβ)，垂直方向是 (cosβ, sinβ)。对相对坐标 (Δx,Δy)=(x0-X, y0-Y) 做旋转：

```
u = Δx·sinβ - Δy·cosβ
v = Δx·cosβ + Δy·sinβ
```

因为 X,Y 是独立同方差的各向同性高斯分布，旋转变换后 (U,V) 仍然是独立同分布 N(0,σ²)（各向同性分布的旋转不变性）。于是"落点是否在旋转矩形内"这个几何条件，在 (u,v) 坐标下变成简单的轴对齐矩形判据：|u|≤L/2 且 |v|≤W/2。**这个变换的价值在于：它把"β 任意"这个看似复杂的维度直接消掉了——只要瞄准点相对艇体中心的局部坐标 (u0,v0) 确定，命中概率的解析式与 β 无关。** 这是我在写代码前先纸笔推导出来的，然后专门写了一段代码验证这个"β 不敏感性"是否真的成立（见下节，结果完全吻合）。

### 4.2 三情形的布尔化简

沿用第二节的记号：
- I_in = 1{|u0|≤L/2, |v0|≤W/2}（落点是否在水平投影内，这里 u0,v0 是瞄准点相对艇体中心的局部坐标）
- Z_top = Z - H/2（艇顶面深度）
- A = I_in ∧ (d ≥ Z_top)
- B = 1{dist((x0,y0,d), Box) ≤ R}，其中 dist 是点到长方体的标准欧氏距离：

```
dist = sqrt( max(|u0|-L/2, 0)² + max(|v0|-W/2, 0)² + max(|d-Z|-H/2, 0)² )
```

题目三种情形对应：情形(1)=A；情形(2)= I_in ∧ ¬(d≥Z_top) ∧ B（此时 A 为假，因为 d<Z_top 破坏了 A 的第二个条件）；情形(3)= ¬I_in ∧ B（此时 A 自动为假）。

于是 情形(2)∪情形(3) = ¬A ∧ B（不论 I_in 取值如何，只要 A 为假、B 为真，就落在这两种情形之一）。

用布尔恒等式 A ∨ (B∧¬A) = A∨B，得到最终统一表达式：

**命中 ⟺ A ∨ B**

这一步化简是我认为本题最关键的"建模技巧"，它把题目文字描述的三条分支规则，压缩成两个可以直接写进概率积分的事件，为后面问题1的解析分解、问题2/3的数值优化都打下了基础。

### 4.3 问题1：证明最优瞄准点在原点

**命题**：当 X,Y 独立同分布 N(0,σ²)（各向同性）时，对任意固定的凸对称（关于某中心对称）目标区域，命中概率关于瞄准点位置的函数在"瞄准该区域的对称中心"处取得最大值。

**初等证明（针对纯 I_in 核心项，不含杀伤半径修正）**：因为 (U,V) 独立，P(I_in) = P(|u0-U|≤L/2)·P(|v0-V|≤W/2)（这里用 U,V 表示旋转后的艇体位置分量，u0,v0是瞄准点局部坐标，与上文顺序略有不同但等价，写代码时我统一用"艇体位置为随机变量、瞄准点固定"的视角）。对第一个因子求导：

```
d/du0 [Φ((u0+L/2)/σ) - Φ((u0-L/2)/σ)] = [φ((u0+L/2)/σ) - φ((u0-L/2)/σ)]/σ
```

标准正态密度 φ 是偶函数、在正半轴单调递减，令导数为零需要 φ((u0+L/2)/σ)=φ((u0-L/2)/σ)，因为 u0+L/2 > u0-L/2，两者要有相同密度值只能是关于0对称，即 u0+L/2 = -(u0-L/2)，解得 **u0=0**。同理 v0=0。所以核心项在瞄准原点时取最大。

**含杀伤半径修正的一般证明**：把 B 的判据写成"点 p 到艇体的距离 ≤ R"，这等价于"p 落在艇体做 Minkowski 和膨胀 R（即长方体四边外扩 R、四角变成半径 R 的圆弧）之后的'圆角矩形'区域内"。这个圆角矩形依然是关于艇体中心对称的凸集。根据 Anderson (1955) 关于对称单峰密度函数在凸对称集合上积分的经典不等式（对称、拟凹的密度函数在原点取峰值，则该密度函数沿任意方向平移后在凸对称集合上的积分关于平移量单调不增），可以推出：瞄准原点仍然是全局最优。我在代码里用网格扫描做了数值验证（见下节），偏移量越大命中率单调下降，与该定理预测一致。

### 4.4 问题2/3：只能数值求解的部分

深度 Z 是截尾正态时，P(Z≤d+H/2) 涉及截尾正态的CDF，虽然有闭式，但完整的命中概率还包含 B 项里 (u0,v0,Z) 的联合三维几何积分（长方体膨胀R之后的圆角长方体与三维正态密度的卷积），**没有初等闭式解**，所以问题2的最优引信深度 d*、问题3的最优阵列间距 (sa,sb,d)，都必须用数值方法（蒙特卡洛+网格搜索）求解。这是我在推导过程中确认"哪里能解析、哪里必须数值"之后做的方法选择，不是偷懒不推了，而是这类截尾分布+圆角长方体的联合积分本身就不存在初等原函数。

### 5.2 问题1：发现"核心项占比很小"的问题，转向完整三情形模型（p1b_correction.py）

```python
P_A = I_in.mean()
P_B_extra = B_only.mean()   # 落点在投影外但仍在杀伤半径内
P_total = P_A + P_B_extra
```

**真实输出（N=3000万）：**
```

P(仅靠定深引信杀伤半径补充命中的概率, 即case3贡献) = 0.06230656666666667
P_total (真实完整模型) = 0.08376676666666666
相对提升 = 290.33544266440515 %
```

这就是前面第二节提到的"发现问题"的那一步：补充项是核心项的近3倍。同时验证了 d 只要落在 [137.5,162.5]（艇顶面到艇底面深度范围）内，结果完全不变（因为此时竖直方向的距离恒为0）：
```

d= 150.0 -> P_A+P_extra = 0.083767  (dz=0.0)
d= 162.5 -> P_A+P_extra = 0.083767  (dz=0.0)
d= 170.0 -> P_A+P_extra = 0.078489  (dz=7.5)     <- 超出艇体范围，开始下降
d= 190.0 -> P_A+P_extra = 0.021460  (dz=27.5)    <- 退化回核心项
```
这组数据也直接证明了"最优引信深度是一个区间 [h0-H/2, h0+H/2]，而不是单点"，且一旦超出这个区间，命中率会明显下降，直至退化为只剩核心项。

### 5.8 问题3：验证"有效尺寸=艇体尺寸+2R"的解释性规律

既然问题1告诉我们杀伤半径让艇体的"有效宽度"从 W 膨胀到 W+2R，我猜测问题3里跨航向间距 sb 的最优值也应该接近 W+2R = 20+2×20 = 60。数值搜索结果 sb*≈60~62，与这个猜测几乎精确吻合。

为了检验这不是巧合，换一个杀伤半径 R=10（此时预测 sb*≈W+2R=40）重新搜索：
```
R=10m 敏感性检验 (预测 sb*约 W+2R=40):
sa=90,sb=40 -> P=0.1763
sa=100,sb=40 -> P=0.1831   <- 各sa组里sb=40都是峰值
sa=110,sb=40 -> P=0.1871
```
确实在 sb=40 附近取得每组的峰值，与"sb* ≈ W+2R"的预测一致。这是我认为本题最有意思的一个结构性结论：**跨航向的最优阵列间距，本质上是杀伤半径膨胀后的艇体有效宽度**，而不是简单套用艇体自身的宽度。

（sa方向的最优值比"L+2R"的朴素类比要小一些——125~130 vs 140，这是因为沿航向的"边带"贡献本来就比跨航向边带小得多（问题1分解里左右边带0.0078 vs 上下边带0.0423），加上阵列在两个方向上同时展开后，相邻弹药在对角线方向上的杀伤区域会产生我未做严格解析、但数值上确认存在的重叠效应，使得沿航向不需要拉得像"L+2R"那么开。这一点我在论文里如实说明是"数值观察到的现象，未给出完整解析证明"，没有过度声称。）

## 六、遇到的问题与调整（汇总）

1. **最大的一次调整**：最初假设"杀伤半径修正项是小量，可以只用核心接触命中项近似"，做了蒙特卡洛之后发现修正项反而是核心项的3倍左右，果断放弃了这个简化，改用完整三情形模型，并补做了解析分解来交叉验证，确保这不是代码bug（两种独立方法结果一致，相对误差0.04%）。
2. **问题3网格搜索范围不够大**：第一次粗网格把 sb 上限设在36，结果最优值卡在边界上，说明真实最优在更大的sb——这是数值优化中常见的"边界解陷阱"，处理方法是扩大搜索范围重新搜，直到最优点落在网格内部而不是边界上，才能确认找到的是真正的局部/全局最优而不是网格边界的人为限制。
3. **蒙特卡洛噪声与"真最优值"的区分**：在细粒度网格里（比如问题1瞄准点偏移x0=0和x0=10），偶尔会出现"偏移后概率反而略高于不偏移"的反直觉数值，这是蒙特卡洛统计噪声，不是模型有问题——我在文中专门标注这类情况在置信区间内，而没有把噪声波动误当作真实结论去解读，这是使用蒙特卡洛方法时必须保持的审慎态度。

## 七、自我批判：这份解答的局限，如果重来会怎么改进

1. **"杀伤半径内必然摧毁、半径外必然安然无恙"是本题（题目原文语义下）合理但仍是简化的假设**。真实深弹杀伤概率通常随距离连续衰减（如高斯型或指数衰减的毁伤概率密度），如果题目/后续研究给出这类衰减函数，模型可以直接把 B 这一项的0/1指示函数换成一个连续的毁伤概率函数，积分框架完全不用变，只是从"事件的示性函数"换成"条件毁伤概率"，是一个自然的扩展方向，我在论文"模型评价与推广"里写明了。
2. **问题3的阵列间距只搜索了"3×3矩形网格、行列间距各自独立可调"这一种阵列拓�扑**，题目图2本身也是暗示这种拓扑，但没有证明这就是"9发弹在任意几何排列下的全局最优阵型"（比如六边形密堆积排列在覆盖类问题里有时优于矩形排列）。受限于时间和题目图示的引导，本文没有跳出矩形阵列去做更大范围的拓扑搜索。
3. **sa 方向最优值的物理机制没有给出严格解析证明**，只是数值观察 + 定性解释（对角重叠效应），这是本文诚实标注的一处"知道现象、未完全讲清楚机理"的地方，而不是假装严格证明了。
4. **蒙特卡洛精度**：问题2、3使用的样本量在 400万~2000万之间，对应的95%置信区间半宽大致在0.01%~0.05%这个量级，对于寻找"最优点附近平坦区域的具体最大值"是够用的，但如果要非常精确地区分比如 sa=125 和 sa=130 谁更优（两者相差不到0.1个百分点），现有样本量下这个差异已经落在噪声范围内，只能说"最优解位于一个平坦邻域"，无法给出更精确的单点判定——这一点我在论文里如实注明为"平坦最优邻域"而不是给一个假装精确到个位数的点估计。
5. **没有考虑深弹本身的投放散布误差**。这是题目条件本身决定的（题目只描述了潜艇定位误差），但现实中飞机投弹本身也有落点散布，如果题目/后续要求加入这个误差源，需要把 (x0,y0) 从确定性决策变量改成有自己分布的随机变量，模型框架上是可以扩展的（只需把瞄准点也变成随机变量、目标函数变成对瞄准点分布参数的优化），但这超出了本题题面给定的范围。
