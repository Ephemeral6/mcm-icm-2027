# 必学模型清单（C 题导向，按优先级排序）

> 从 [README](README.md) 拆出的独立清单，加入了第二轮调研的 O 奖论文实证依据。学法（社区共识）：公式 → 适用场景 → 手推一遍原理 → 用真题数据跑一遍模板代码。
> 每个模型的**选用策略**（什么时候用、易错点、搭配套路、评委态度）见 [12-模型与算法策略](../12-模型与算法策略/README.md)——本清单管"学什么"，那边管"怎么选"。

## 优先级 P0（赛前必须人人能讲清、代码库必备）

| 模型 | 用途 | Python 实现 | O 奖实证 |
|---|---|---|---|
| 多元线性回归 / 逻辑回归 | 基线模型、解释性分析 | scikit-learn / statsmodels | 常见 |
| 随机森林 / **XGBoost** | 预测 + 特征重要性 | xgboost | 2024 C O 奖用 XGBoost 做因子贡献分析 |
| **SHAP 可解释性** | 给 ML 模型配解释层 | shap | 2024 C O 奖用了，评委吃这套 |
| K-means 聚类 + PCA | 分组、降维 | scikit-learn | 2024 C O 奖用于迁移学习分组 |
| 时间序列：ARIMA、指数平滑/EWMA | 趋势预测、动量量化 | statsmodels | 2024 C O 奖用 EWMA 量化势头 |
| **敏感性分析** | O 奖论文标配独立章节 | 自写（参数扰动+结果对比） | 连续三年题面点名 |
| 不确定性量化 | 置信区间、Bootstrap、预测区间 | scipy / 自写 | 2025/2026 题面明示要求 |

## 优先级 P1（至少建模手要会，代码库最好有）

| 模型 | 用途 | 实现 |
|---|---|---|
| 熵权法 / TOPSIS / AHP | 评价与权重（构造综合指数） | 自写模板（简单） |
| 灰色预测 GM(1,1) | 小样本趋势 | 自写模板 |
| 马尔可夫链 | 状态转移（排名变动、淘汰过程） | numpy 自写 |
| LSTM（会调用即可） | 序列预测 | PyTorch/Keras；2024 C O 奖主力模型 |
| 贝叶斯估计 / MCMC 入门 | 隐变量反演（2026 C 粉丝票就是隐变量） | PyMC（可选，难度较高） |

## 优先级 P2（了解思想，赛中现学也来得及）

线性/非线性规划（scipy.optimize）、动态规划、多目标优化、遗传算法/模拟退火、图论基础（networkx）、排队论、元胞自动机、SEIR 类微分方程。

## 通用工序（比模型更重要，全员掌握）

1. **数据清洗**：缺失值策略（删/补/标记）、异常值、类型转换——O 奖论文有专门 Data Cleaning 章节
2. **EDA**：分布、相关性热图、分组对比——O 奖论文的 Data Overview 章节
3. **模型比选**：同一问题至少两个候选模型对比后论证选择（O 奖套路：RNN/GRU/LSTM 比完选 LSTM）
4. **验证**：留出集/交叉验证、误差指标选择与解释
5. **结果呈现**：每个模型输出至少配一张图 + 一段"这说明了什么"
