# 05 备赛路线图（2026.7 → 2027.2）

> 月度计划为综合社区经验与你们队情况的**建议方案**（【经验·未验证】性质），模型清单与工具链来自多个获奖者攻略的交集。

## 0. 总原则

- 你们目标题型是 **C 题（数据题）**，路线围绕"数据分析 + 预测建模 + 英文论文"展开，不必全面铺开学所有模型。
- 社区反复强调的第一学习法：**精读 O 奖论文比刷教材重要**（有 O 奖队伍赛前把近五年优秀论文全部读完整理）。
- 每个模型按三步学：公式 → 适用场景与问题类型 → 手推一遍原理（社区共识学法）。
- 训练 AI 协作能力和训练数理能力同等重要——AI 在哪些环节可靠、哪些环节容易出错，见 [14-AI在数学建模中的应用与局限](../14-AI在数学建模中的应用与局限/README.md)，备赛时把"审查 AI 输出"练成一项和"建模"并列的技能。

## 2. 必学模型清单（C 题导向，社区攻略交集）

**评价/权重类**：层次分析法（AHP）、熵权法、TOPSIS、主成分分析（PCA）
**预测类（C 题核心）**：线性/多元回归、逻辑回归、时间序列（ARIMA）、灰色预测 GM(1,1)、马尔可夫链、SEIR 类微分方程（了解）
**机器学习（C 题高频）**：随机森林、XGBoost、SVM、K-means 聚类、神经网络（会用即可）；重点练**特征重要性分析与模型解释**——评委反感纯黑箱（见 06）
**优化类（了解即可）**：线性/非线性规划、动态规划、多目标优化、遗传算法/模拟退火
**通用工序**：数据清洗与缺失值处理、相关性分析、敏感性分析（O 奖论文标配）、误差/稳健性检验
