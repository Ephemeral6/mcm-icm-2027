# 02 预测类模型策略

> 适用信号总纲：题目在问"**未来是多少/趋势如何/何时发生**"。
> C 题相关度：★★★★★（时序是 C 题第一高频工具：2018–2021 年 8 篇 O 奖中 5 篇用时间序列【经验·单一来源】；2024/2025 连续两年 C 题 O 奖主模型都是时序类）

## 0. 快速决策表（数据条件驱动【已验证原则】）

| 数据条件 | 选它 |
|---|---|
| 能写出内部机理（增长率、转移规律） | 微分/差分方程（机理建模） |
| 样本极少（4–20 点）+ 近指数趋势 + 短期 | 灰色 GM(1,1) |
| 单变量、平稳（或差分后平稳）、中等长度 | ARIMA |
| 明显季节性 | SARIMA / Prophet |
| 多个相互关联变量一起预测 | VAR |
| 长序列 + 长期依赖 + 数据量大 | LSTM |
| 状态离散、只依赖当前状态 | 马尔可夫链 |
| 因变量与自变量近似线性 | 回归（**别上神经网络**） |

**第一分诊【经验·多来源】**：能否用数学语言刻画演化机理？能 → 机理建模（微分方程）；不能 → 数据驱动（统计/ML）。混合场景（机理已知+参数未知）用机理模型+数据拟合参数。

## 1. 逐模型策略卡

### 回归家族（线性/多元/逻辑/岭/逐步）【经验+O 奖实证】
- **信号**：解释"哪些因素影响 Y 且方向如何"；线性关系场景的第一选择。
- **诊断纪律**：VIF>10（严格 5）判多重共线性 → 改岭回归/逐步回归；残差分析写进论文。
- **二分类**（如"哪些国家将首夺奖牌"）：逻辑回归（解释强）或随机森林（非线性），配蒙特卡洛估计突破概率【经验·2025 C 复现文】。

### ARIMA 家族【已验证要点】
- **信号**：单变量、中等长度序列。**前提是平稳**——ADF 检验，不平稳先差分。
- **常见误用**【已验证】：拿 ARIMA 预测受政策/新闻扰动的非平稳数据（股价类）。
- **变体选择**：季节性 → SARIMA；多变量互相影响 → VAR。
- **库**：statsmodels。

### 灰色预测 GM(1,1)【已验证·学术印证充分】
- **信号**：小样本（最少 4 点）、近似指数增长、**中短期**预测。
- **硬流程**（不做=失分）：建模前**级比检验**（不过则序列平移），建模后**后验差比 C 值检验**，都通过才能用于预测。
- **禁区**：长期预测、纯波动序列。
- **库**：numpy 手写（30 行）。

### 马尔可夫链【经验】
- **信号**：状态离散、未来只依赖当前状态（排名迁移、市场份额、淘汰过程）。
- **易错**：状态划分随意、转移矩阵样本不足；"只依赖当前状态"这一假设要论证。

### LSTM / 深度时序【O 奖实证】
- **信号**：长序列、长期依赖、数据量足够；C 题近两年 O 奖主力（2024 比选 RNN/GRU/LSTM；2025 双通道 LSTM + HostFlag/PrepFlag 特征工程）。
- **纪律**：必须做**比选**（和 ARIMA/简单基线对比并给指标），必须配可解释层（特征重要性/SHAP），否则就是评委点名的黑箱；特征设计要对应数据生成机制（2025 O 奖为主场效应单独开通道）。
- **库**：PyTorch / Keras。

### Prophet【经验】
- **信号**：业务型序列（多周期季节性、缺失值多、有节假日效应），自动化程度高。
- **定位**：快速基线；论文中作为对照模型比作为主模型更稳妥。

## 2. 不确定性量化（C 题硬要求，连续三年题面点名）

- **Bootstrap 预测区间**【O 奖实证】：重复抽样训练多模型，取预测分布中间 95%——2025 C 题 O 奖 2500759 的做法；非参数、不依赖正态假设。
- **分位数回归**：直接建模预测区间上下界【经验·复现文推荐】。
- **时间序列交叉验证**：按时间（奥运按届次）划分训练/测试，不能随机打乱。
- **检验指标要贴合问题**：2025 O 奖用加权 MAE（奖牌大国误差加权）——从文献/问题结构出发定指标，不自造。

## 3. 本类论文写法要点

1. 主模型 + 基线模型对比是标配（LSTM vs ARIMA vs 朴素外推，给同一指标）；
2. 预测必须带区间，单点预测在 C 题基本失分；
3. 外推边界要自我声明（灰色预测别外推 3 步以上、LSTM 别预测训练分布外的场景）。
