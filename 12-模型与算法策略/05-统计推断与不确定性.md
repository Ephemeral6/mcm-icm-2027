# 05 统计推断与不确定性策略

> 适用信号总纲：题目在问"**这个效应真的存在吗/结论多可靠/区间是多少**"。
> C 题相关度：★★★★★——敏感性分析与不确定性量化是**已验证的评审硬要求**（官方 Tips + 历年评委评论 + 连续三年 C 题题面点名），本文件是全仓库"防失分"浓度最高的一份。

## 1. 敏感性分析操作手册（已验证策略 + 可操作模板）

**原则**【已验证·三票】：
- 不要给本应取值范围的参数指定固定"合理"值——最终分析必须展示**结果如何随该参数变化**；
- **优先分析无法准确测量/估计的参数**，不必全参数扫描；
- 历年 O 奖论文绝大多数含独立的敏感性分析章节【经验·多来源一致】。

**两种做法（按需选用）**：

### 局部法：逐参数扰动【经验·标准做法】
1. 挑出 2–4 个最不确定的参数；
2. 在基准值附近取扰动网格（惯例 ±20%，如 r ∈ 0.8–1.2 倍基准）；
3. 重算模型输出，画"参数—输出"曲线或龙卷风图；
4. 结论句式："输出对参数 X 的弹性为__，在扰动范围内核心结论（排序/符号/量级）不变/在__处反转"。

### 全局法：蒙特卡洛敏感性分析【经验·四步模板】
1. **参数设定**：给每个不确定输入假设概率分布（均匀/三角/正态，说明依据）；
2. **随机抽样**：从联合分布抽大量样本（≥1000 组）；
3. **批量计算**：每组样本跑一遍模型得输出；
4. **统计分析**：输出分布直方图 + 均值/区间 + 输入-输出相关性排序（哪个参数贡献最大不确定性）。
- 注意自行补充局限声明：计算量大、依赖分布假设【经验】。

## 2. 不确定性量化工具箱

| 工具 | 信号 | 要点 |
|---|---|---|
| **Bootstrap**【O 奖实证】 | 给任何统计量/预测配区间，不依赖正态假设 | 重抽样≥1000 次；2025 C O 奖用"Bootstrap 训练多模型取中间 95%"出预测区间 |
| 蒙特卡洛模拟 | 解析解难求 + 存在随机性（如"突破首奖概率"） | 报告模拟次数与收敛性；2025 C 复现文用它估计二分类突破概率 |
| 分位数回归 | 直接建模预测区间上下界 | 非参数区间的另一条路 |
| 置信区间/预测区间 | 一切"预测未来值"的问题 | C 题单点预测≈失分；区分置信区间（参数）与预测区间（未来观测） |

## 3. 假设检验与相关分析【经验】

- **用途信号**：验证"效应是否存在"类子问——2025 C 题"伟大教练效应"的正确打开方式就是**从历史数据找教练更替前后的成绩突变证据**（数据驱动检验），而不是直接假设效应存在【经验·复现文思路】；
- 常用：t 检验/方差分析（组间差异）、卡方（分类关联）、相关系数（先看散点图再报数）；
- 纪律：报告 p 值同时报告效应量；多重检验要说明；"相关≠因果"在政策建议前必须自我声明。

## 4. 贝叶斯方法【经验 + O 奖实证】

- **信号**：隐变量估计（2026 C 粉丝票就是隐变量）、小样本融合先验、在线更新；
- **O 奖实例**：2024 C 题 O 奖 2403774 用**贝叶斯在线拐点检测**定位势头转折区间，再接 TCN 精确预测——"贝叶斯定位+ML 精化"的组合示范；
- 成本提示：MCMC 实现门槛高（PyMC），赛中量力而行；共轭先验的简单贝叶斯更新可手写。

## 5. 模型检验指标策略【已验证原则 + O 奖实证】

- **用文献中的标准指标，不要自造**【已验证·三票】；
- 指标要贴合问题结构：2025 C O 奖用加权 MAE（奖牌大国误差加权）——"选这个指标因为__"本身要写一句论证；
- 时序问题用时间切分交叉验证（按届次/年份），随机打乱是常见硬伤。

## 6. 本类论文写法要点

1. 敏感性分析独立成章（放在结果之后、优缺点之前），至少一图一表一结论句；
2. 每个预测数字旁边都应该有区间或误差指标；
3. 诚实策略【已验证·评委实证】：2003 年 O 奖论文坦承未达题目 90% 覆盖率要求并解释缺陷照样拿 O 奖；而多支本可获 O 奖的队因未引用网上资料被降级——**承认局限加分，隐瞒来源降级**。
