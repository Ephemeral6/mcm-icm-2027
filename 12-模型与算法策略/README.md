## 1. 最重要的结论：评委要什么（已验证·官方）

COMAP 官方 Tips 原文定义"最佳论文"：**超越题目要求的完整逻辑分析**——有论证的假设、建模过程、结果、优缺点自评、敏感性分析、结论。官方文件通篇强调分析深度与结构化自我批判，**从未奖励方法复杂度**。

2024 年 C 题评委评论（转述，多来源一致）进一步印证：
- 评委明确**不认可"黑箱"式使用机器学习**——直接把 ML 输出合理化为结论、不讨论参数选择原因的做法，即使事后补交叉验证也不被认可；
- 评委推荐的路径：**先用最少假设可用的方法，再逐步放宽假设提升模型**；
- 走过场式假设（"假设数据真实准确"）不被看好；好假设 = 数学化陈述 + 合理解释 + 敏感性分析配合。

**一句话方针：简单模型 + 深入解释 + 敏感性分析 > 复杂模型堆砌。**

**国赛评委在同一件事上说得更直接**（2024 年组委会公开会议纪要，已验证·官方，见 [13-国赛CUMCM/官方评阅内幕](../13-国赛CUMCM/官方评阅内幕(2024年会议纪要).md) 第 1 节）：官方评阅标准原文"从问题出发，拒绝套路化！"，并专门用一页 PPT 点名批评"n 板斧"现象——不管题目是什么都套用同一份固定模型清单（AHP/模糊综合/熵权法/回归/PCA/聚类/灰色/遗传算法/神经网络/深度学习……），批评其"一知半解（甚至不知不解）、牵强附会"。**这份被点名的清单几乎覆盖本板块全部内容**——不是说这些模型不能用，而是必须真的理解为什么用、论文里要体现理解，国赛和美赛评委在这一点上的立场完全一致。

## 2. 模型选用一级框架：问题特征 → 模型类别

先分诊到五大类（社区稳定共识，多来源交叉印证【经验】），再进对应文件选具体模型：

| 问题特征（题目在问什么） | 类别 | 去哪个文件 |
|---|---|---|
| "评价/排序/选出最优方案/公平性" | 评价决策 | 01 |
| "预测未来值/趋势/什么时候发生" | 预测 | 02 |
| "怎么安排/分配/调度使目标最优" | 优化 | 03 |
| "分类/分组/识别模式/哪些因素重要" | 分类与机器学习 | 04 |
| "检验假设/量化不确定性/相关性" | 统计推断 | 05 |
| "网络/传播/路径/演化仿真" | 图论与仿真 | 06 |

**二级分诊的关键启发式**（各文件展开）：
- 能用微分方程写出内部机理 → 机理建模；写不出 → 数据驱动【经验】
- 线性关系用回归，别上神经网络（神经网络对线性拟合反而差）【经验·知乎高赞】
- 小样本（可少至 4 点）+ 近指数趋势 + 短期 → 灰色预测【已验证·学术印证】
- 多阶段决策 → 动态规划；凸优化 → 梯度法；非凸 → 启发式【经验】
- 高维特征 → PCA 降维；找隐变量 → 因子分析【经验】

## 3. 三条全局策略（已验证程度最高的实战建议）

1. **多模型互证**【已验证·O 奖队指南+无反证】：优先构建两个基于**不同假设**的竞争模型（典型：解析模型 + 数值模型），结论一致且与文献相符 → 相互验证。避免单一庞大仿真（monolithic simulation，易变成无根基的 ad hoc 模型）。
2. **由简到繁**【经验·多来源一致】：先建最简模型跑通全链路，再逐步放宽假设复杂化——这同时是评委推荐的 ML 使用路径。
3. **敏感性分析是硬要求**【已验证·官方+评委评论】：不要给本应取值范围的参数拍一个"合理"固定值——必须展示结果随参数的变化；优先分析**无法准确测量/估计**的参数（不必全参数扫描）。评价自己的解要用**文献中的标准指标**，不要自造指标。

## 4. O 奖论文实际用什么（实证数据点）

- **2024 C 题 O 奖 2406324**：EWMA → RNN/GRU/LSTM 比选 → XGBoost 因子分析 → PCA+K-means → SHAP → 敏感性分析（详见 [10-O奖论文研究](../10-O奖论文研究/README.md)）
- **2025 C 题 O 奖 2500759**【经验·社区复现文】：数据清洗/特征构造 → PCA 降维 → 双通道 LSTM（HostFlag 主场/PrepFlag 筹备期特征）→ XGBoost+Bootstrap 出 95% 置信区间 → 加权 MAE（对奖牌大国误差加权）做检验
- **2024 C 题其他 O 奖**（评委评论转述）：2403774 用"贝叶斯在线拐点检测+TCN+自研 MSTCN"；2401919 用 **PCA-TOPSIS**（评委点名：传统 TOPSIS 依赖主观权重，PCA 是客观替代）
- 2018–2021 年 O 奖预测类统计【经验·单一来源】：8 篇 O 奖中 5 篇用了时间序列分析——时序是 C 题第一高频工具
- **重要的负面结论**【已验证】：公开渠道**不存在**严格的"O 奖模型使用频率统计"（现有文章都是清单式汇总）。想要精确画像需自行抽样 O 奖论文逐篇编码——这本身是备赛期的高价值练习。

## 5. 被证伪的流传说法（引用前必看）

以下说法在本轮调研中被 3 票否决（0-3），**不要写进论文或用于决策**：

| ❌ 流传说法 | 实际情况 |
|---|---|
| "优化类是美赛最常见题型" | 无证据；近年 C 题（数据）才是选题人数第一 |
| "评价模型是美赛应用最多的模型" | 无证据支撑该断言 |
| "熵权法完全客观、避免一切主观干扰" | 熵权忽视决策者经验与指标实际含义，对极端值敏感；学界建议 AHP+熵权组合 |
| "多目标规划标准解法=加权转单目标+遗传算法" | 加权法只是多种标量化方法之一（还有 ε-约束、NSGA-II 直接求 Pareto 前沿等） |
| "C 题 2016 年新增且数据包 100MB+" | 与官方材料不符 |
