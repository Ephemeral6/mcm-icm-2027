# 04 机器学习与可解释性策略

> 适用信号总纲：题目在问"**分类/分组/哪些因素重要/复杂非线性预测**"。
> C 题相关度：★★★★★（近年 C 题 O 奖标配，但**评委反黑箱的态度同样是已验证的**——本文件的核心就是怎么用 ML 不踩雷）

## 0. 先读红线：评委对 ML 的真实态度

- 2024 年 C 题评委评论（多来源转述一致）：**不认可黑箱式 ML**——直接把模型输出合理化为结论、不讨论参数/特征选择原因，即使事后补交叉验证也不认可；
- 评委推荐路径：**少假设的简单方法起步 → 逐步放宽假设**；
- 官方 Tips【已验证】：评奖看的是有论证的假设、过程、优缺点自评、敏感性分析——不是方法先进性。
- **结论：每个 ML 模型必须配三件套——为什么选它（比选）、它学到了什么（可解释层）、结论稳不稳（敏感性/检验）。**

## 1. 快速决策表

| 问题特征 | 选它 |
|---|---|
| 表格数据预测/分类，要特征重要性 | 随机森林（稳、免调参）或 XGBoost（强、要调参） |
| 线性可分或样本少的分类 | 逻辑回归 / SVM |
| 无标签分组（国家/选手/项目聚类） | K-means（配肘部法/轮廓系数定 K） |
| 高维变量压缩、相关性强 | PCA |
| 探索隐变量结构 | 因子分析【经验·与 PCA 的区分】 |
| 解释任何黑箱模型 | SHAP（全局+单样本归因） |

## 2. 逐模型策略卡

### 随机森林 / XGBoost【O 奖实证】
- **信号**：中小规模表格数据的非线性预测/分类；C 题最常用 ML（2024 O 奖用 XGBoost 做因子贡献；2025 O 奖用 XGBoost+Bootstrap 出区间；社区复现文普遍 RF/XGBoost/KNN/BP 四件套）。
- **标准搭配**：**XGBoost + SHAP**——预测力和解释力兼得；或 RF + 特征重要性（简版）。
- **纪律**：训练/测试划分方式要写明（时序数据按时间切，不能随机打乱）；给基线对比；报告过拟合检查。

### SVM / KNN【经验】
- SVM 信号：样本少、维度高、边界复杂的分类；核函数选择要说明理由。
- KNN 信号：懒人基线，适合当对照模型；K 与距离度量做敏感性。

### K-means 聚类【O 奖实证】
- **信号**：无标签分组；2024 O 奖用 PCA+K-means 给比赛分组后做迁移学习（组内规律更同质 → 提升预测精度）——"聚类不是目的，是给下游模型分层"的高级用法。
- **纪律**：K 的选择要有依据（肘部法/轮廓系数图进论文）；聚类结果要能被业务解释（每类是什么含义）。

### PCA / 因子分析【O 奖实证 + 经验】
- **PCA 信号**：变量多且相关性高，需降维/去共线性/客观定权。
- **O 奖实例**：2025 O 奖对 70+ 运动项目 PCA 压缩；2024 O 奖 2401919 的 **PCA-TOPSIS**（主成分特征值占比当权重）获评委点名认可——PCA 是"规避主观权重"的正规武器。
- **区分**【经验】：PCA 是压缩信息；因子分析是找隐变量。别混用术语。
- **纪律**：报告累计方差贡献率（≥80–85% 惯例）；主成分要给出解释（载荷最大的原始变量是什么）。

### SHAP 可解释性【O 奖实证】
- **信号**：任何用了树模型/神经网络的场合，都值得加一节 SHAP。
- **产出三图**：全局特征重要性（summary plot）、单特征依赖图、个案归因（force plot）——直接回应评委"模型学到了什么"的质疑。
- **库**：shap（对 XGBoost/LightGBM 原生支持，快）。

## 3. 本类论文写法要点

1. ML 章节的叙事顺序：问题特征 → 候选方法比选（表格对比）→ 选定理由 → 训练细节 → 检验指标 → **SHAP/重要性解释** → 与机理直觉互证；
2. "先简后繁"落到实处：先给逻辑回归/线性基线的结果，再展示 XGBoost 提升了多少——这一张对比表同时完成了比选和论证；
3. 神经网络（BP/LSTM）不是加分项本身，配上解释层和比选才是。
