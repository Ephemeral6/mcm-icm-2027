# 06 图论网络与仿真类策略

> 适用信号总纲：题目在问"**路径/连接/传播/系统随时间演化**"。
> C 题相关度：★★（备而不用；D 题的主战场，E/F 题偶用传播与仿真）
> 可信度说明：本类未进入三票验证名单，内容为抓取素材与通识整理【经验】。

## 0. 快速决策表

| 问题特征 | 选它 |
|---|---|
| 两点间最短路/最优路径 | Dijkstra（单源）/ Floyd（全源） |
| 管道/物流的最大通过能力 | 最大流（Ford-Fulkerson） |
| 网络中谁最重要 | 中心性指标（度/介数/接近/PageRank） |
| 疾病/谣言/技术扩散 | SEIR 族微分方程 |
| 个体规则→群体涌现（交通/火灾/人流） | 元胞自动机 CA |
| 排队/服务系统（窗口数、等待时间） | 排队论 M/M/c |
| 访问所有点的最短回路 | TSP → 规模大时启发式（NP 完全，组合爆炸是正当理由）|

## 1. 策略卡

### 图论基础（最短路/最大流/匹配）【经验】
- 建模关键在**抽象成图**：什么是节点、什么是边、边权是什么——这一段写清楚比算法本身更重要；
- 库：networkx（Dijkstra/Floyd/最大流/中心性全有，不要手写）。

### 网络科学（D 题向）【经验】
- 信号：题目给出关系数据（贸易网、社交网、航线网）；
- 套路：建网 → 度分布/中心性/社区发现（Louvain）→ 网络指标变化的动态分析；
- 图表标配：网络可视化图 + 中心性排名表。

### SEIR 族传播模型【经验】
- 信号：任何"扩散"语义（疫情/舆情/技术采纳）；
- 写法四件套：仓室图 → 微分方程组 → 参数含义表（R0 必须讨论）→ 数值解曲线；
- 参数不确定 → 直接衔接 [05](05-统计推断与不确定性.md) 的敏感性分析（SEIR 的 β、γ 是敏感性分析的经典素材）；
- 库：scipy.integrate.odeint。

### 元胞自动机 CA【经验·O 奖清单出现过】
- 信号：宏观方程写不出、但个体规则清晰的演化系统（森林火灾/交通流 Nagel-Schreckenberg/人群疏散）；
- 纪律：规则表要完整列出；多次随机初始化取统计；网格大小做敏感性；
- 提醒：CA 属于"单一庞大仿真"风险区——记得 [README](README.md) 的多模型互证原则，配一个简化解析模型对照。

### 排队论【经验】
- 信号：到达-服务结构明确的系统优化（收费站/客服/急诊）；
- M/M/c 公式可手算，超出假设（非泊松到达）就转离散事件仿真（simpy）。

## 2. 本类论文写法要点

1. 仿真类结果必须报告：初始化设置、运行次数、随机种子处理、收敛/稳态判断；
2. 涌现结论要配机制解释（为什么这个个体规则导致这个宏观模式）；
3. 传播/仿真模型天然适合"情景分析"（干预 vs 不干预），这是政策建议类小问的标准衔接。
