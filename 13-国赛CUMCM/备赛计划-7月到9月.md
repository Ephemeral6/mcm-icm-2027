# 国赛备赛计划（2026-07-13 → 09-10，约 8 周）

> 参照高校官方集训模式（上海交大培训营：暑期"课程授课"+ 赛前 1–2 周"全仿真模拟"两阶段，单源【经验】）改编为自训方案，并衔接美赛备赛路线（[05-备赛路线](../05-备赛路线/README.md)）。前提：本科组选 C 题（理由见 [近三年真题与题型](近三年真题与题型.md)）。

## 0. 与美赛备赛的关系

国赛不是额外负担，而是把美赛路线里"11 月第一次全真模拟"提前成一场真刀真枪的比赛。两赛通用：数据工作流、模型库、分工协作、AI 留痕习惯。国赛专项补差（美赛路线里没有的）：

1. **规划求解**：国赛 C 题的"决策/优化"味重——线性/整数规划建模 + PuLP/OR-Tools 求解，专项练两周；
2. **中文科技写作**：摘要+正文的中文规范表达（美赛练的是英文）；
3. **可运行代码交付**：国赛要交源程序且"跑不通可能取消评奖"——代码规范必须前置；
4. **MD5 两步交卷流程**：美赛没有的环节，必须赛前演练（见 [赛中流程与提交规范](赛中流程与提交规范.md)）。

## 1. 周计划

### 第 1 周（7/13–7/19）：报名确认 + 定题方向
- [ ] **联系教务处/数学学院确认校内报名**（最高优先级，见 [README](README.md) 第 0 节）
- [ ] 确认三人 8–9 月的时间（暑假谁在校/线上协作方案）
- [ ] 通读本目录全部文件 + 官方三份 PDF；共识：主攻 C 题
- [ ] 装环境：**准备一台 Windows 电脑**（交卷客户端只支持 Windows，Mac 队员注意）

### 第 2–3 周（7/20–8/2）：补短板
- [ ] 规划求解专项：线性/整数规划建模写法 + PuLP 上手，用 2024 C（种植策略，数据已入库 `真题2024/`）练
- [ ] 中文论文模板：按《论文格式规范》搭好 Word/LaTeX 模板（摘要专用页、页码从摘要页起、附录含源程序）
- [ ] 精读 2–3 篇官方优秀论文（dxs.moe.gov.cn，选近三年 C 题）+ 对应官方赛题讲评

### 第 4–5 周（8/3–8/16）：分题实练
- [ ] 用 2023 C（蔬菜定价）做半程练习（48 小时内出主体模型+初稿）
- [ ] 每人完成岗位职责卡的国赛版核对（AI 台账 → "AI 工具使用详情"PDF 的导出流程走一遍）
- [ ] 复盘：数据→建模→求解→写作各环节的工时

### 第 6 周（8/17–8/23）：第一次全真模拟
- [ ] 用 **2025 C（NIPT，数据已入库）** 做 74 小时全真模拟：周四 18:00 开始、周日 20:00 "交卷"，产出完整论文 + 支撑材料压缩包 + AI 详情 PDF
- [ ] 模拟 MD5 流程：生成文件 MD5、"锁稿"后不再打开文件
- [ ] 复盘会：对照官方讲评检查思路偏差

### 第 7 周（8/24–8/30）：查漏 + 二次演练
- [ ] 针对模拟暴露的短板专项补强（常见：优化模型写不规范、中文摘要空泛、代码不可复现）
- [ ] （可选）48 小时压缩版二次模拟，练时间控制
- [ ] **8 月下旬复查 mcm.edu.cn**：2026 版《参赛须知》与 AI 规定是否有更新

### 第 8 周（8/31–9/9）：赛前收尾
- [ ] 冻结模板与代码库；Windows 客户端安装并确认**报名信息中的第一位学生**（只有 TA 能提交）能登录
- [ ] 打印检查：承诺书/编号页流程、纸质版装订要求，问清学校纸质版收取安排
- [ ] **退出所有数模群**，约定赛期不开知乎/CSDN/GitHub（平台浏览禁令，见提交规范）
- [ ] 排 74 小时值班表（比美赛短一天，节奏更紧：D1 晚定题、D2 建模求解、D3 写作合体、D4 白天终校 20:00 交 MD5）

### 赛期（9/10 18:00 – 9/13 20:00）
- 时间表参照 [05/赛中96小时作战手册](../05-备赛路线/赛中96小时作战手册.md) 压缩为 74 小时版；
- 硬节点：**9/13 20:00 前上传 MD5 码**（上传后绝不再打开文件）；**9/13 20:30–9/14 14:00 上传电子文档**；纸质版按学校通知。
