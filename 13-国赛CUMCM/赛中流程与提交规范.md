# 国赛赛中纪律与提交规范（官方原文整理）

> 直接依据本目录 `官方文件/` 内两份官方 PDF 整理：《参赛规则（2026 修订稿，2026-03-01 试行）》《论文格式规范（2026 修订稿）》。本文件全部为官方一手信息。

## 1. 赛中纪律红线（参赛规则原文要点）

1. **严禁与队外任何人交流赛题**（包括指导教师）。主动讨论和**被动接收**讨论信息都算严重违纪。
2. **指导教师赛期零指导**：解释赛题、建议选题、提供资料、改论文都不行——违者按违纪处理，且该教师两年内不得再带队报名。
3. ⚠️ **平台浏览禁令（比美赛严得多，本队最容易踩的雷）**：竞赛期间不得在任何交流平台——原文点名"贴吧、QQ 群、微信群、网络直播间、知乎、小红书、**CSDN、GitHub**等"——**浏览**、发布或讨论与赛题相关的内容。注意"浏览"也算，严重违纪无条件取消评奖。
   - 对本队的含义：赛中查资料只走文献库、教科书、官方数据源；**不要搜"国赛 A 题思路"之类的内容**；用 AI 工具没问题（见下），但别去逛社区。
4. **抄袭无条件取消评奖**：引用任何公开资料（含网上资料）必须正文标注 + 参考文献列出，不得大篇幅照抄。
5. AI 可以用作辅助，但作品的原创性、真实性、准确性责任全在参赛队，须遵守《AI 工具使用规定》（见 [README](README.md) 第 2 节：正文标注 + 参考文献格式 + "AI 工具使用详情"支撑材料；未用也要声明）。
6. 严重典型违纪会被公开通报批评，学校还要作出处理。

## 2. 论文格式（格式规范原文要点）

### 纸质版（要打印装订提交）
- A4，页边距四边 ≥2.5cm，左侧装订，单双面均可；
- 页面顺序：**第 1 页承诺书 → 第 2 页编号专用页 → 第 3 页摘要专用页 → 第 4 页起正文**；
- 摘要（含标题和关键词，不用译英文）原则上不超过一页；页码从摘要页记"1"，页脚中部；
- **正文不超过 30 页，不要目录**；
- 附录页数不限，但必须打印并与正文装订；附录必须包含：支撑材料文件列表 + **全部完整、可运行的源程序代码**（含 EXCEL/SPSS 交互命令）。缺程序、程序跑不通、或运行结果与论文不符 → 可能取消评奖。真没用程序要写明"本论文没有用到程序"；
- 摘要页、正文、附录任何地方不得出现身份/学校/赛区信息；
- 字体字号行距颜色不统一要求（赛区可以加要求，注意本赛区通知）。

### 电子版（通过 cumcm.cnki.net 提交两个文件）
1. **参赛论文**：单个 PDF 或 Word（建议 PDF），≤20MB，不压缩；**电子版第一页必须是摘要专用页**——承诺书和编号页不放进电子版；内容与纸质版（含附录）完全一致。
2. **支撑材料**：一个 RAR 或 ZIP，≤20MB。至少包含：所有可运行源程序、自主查阅使用的数据资料（赛题自带数据除外）、大篇幅中间结果图表；再加上 AI 规定要求的"AI 工具使用详情"PDF。文件列表要写进论文附录；真没有支撑材料要在附录注明"本论文没有支撑材料"。**支撑材料与论文内容不符 → 可能取消评奖**；所有文件里都不能有身份信息。

### 违规后果
不符合格式规范 = 违反竞赛规则，可能取消评奖资格。

## 3. 与美赛提交的差异对照（备赛时防混淆）

| 维度 | 国赛 | 美赛 |
|---|---|---|
| 语言 | 中文，摘要不用译英 | 英文 |
| 页数 | 正文 ≤30 页（不含附录），**不要目录** | 全文 ≤25 页（含目录/附录/代码） |
| 首页 | 电子版第一页=摘要页（纸质版前面还有承诺书+编号页） | 第一页=Summary Sheet |
| 代码 | **必须交可运行源程序**（附录+支撑材料），跑不通可能取消评奖 | 不强制交码 |
| AI 披露 | 正文标注+参考文献+支撑材料"AI 工具使用详情"PDF | 行内引用+文末 Report on Use of AI Tools |
| 赛中上网 | 禁止在社区平台浏览赛题相关内容 | 无此禁令（只禁与人讨论） |
| 提交 | cumcm.cnki.net，论文+支撑材料两个文件，各 ≤20MB | forms.comap.org，单 PDF ≤25MB |
