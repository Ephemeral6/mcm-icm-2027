> 2025/2024 年题名与题面**直接取自官方赛题包**（本目录 `真题2025/`、`真题2024/` 内为官方原件）；2023 年题名经教育部"中国大学生在线"官方讲评页确认。官方真题下载：mcm.edu.cn "历年竞赛赛题"栏目（1992–2025 全部免费）。

## 0. 组别与选题

- 竞赛设**本科组**（所有大学生可参赛）和**高职高专组**（仅高职高专及职业本科学生，也可选参加本科组），研究生不得参赛（2026 年第一次通知 + 章程原文）。
- 历年惯例：**本科组做 A/B/C 题，高职高专组做 D/E 题**（从历年获奖名单按题分组可印证；正式以 9 月选题界面为准）。
- 即：你们作为本科队，**实际只在 A/B/C 三题里选**——比美赛六选一的选题压力小。

## 2. 题型规律与选题预判

1. **A/B 题偏机理**：几何、物理、运动学场景，数学功底要求高——与美赛 A/B 定位相似，不适合你们。
2. **C 题连续三年都是"真实数据 + 决策/优化"**：蔬菜定价（2023）→ 种植策略（2024）→ NIPT 时点（2025）。与美赛 C 题的区别：国赛 C 题的"优化/决策"味更重——美赛 C 更多是"预测+解释"，国赛 C 通常要在数据分析之上再解一个明确的优化/决策问题。
3. **对你们的结论：国赛选 C 题**，与美赛 C 题备赛高度复用（数据清洗、EDA、预测、不确定性），需要专项补的是**规划求解**（线性/整数规划建模 + Python 求解器，见 [12/03-优化类](../12-模型与算法策略/03-优化类.md)）。
4. 练兵素材优先级：2025 C（NIPT，数据已入库）→ 2024 C（种植策略，数据已入库）→ 2023 C（蔬菜定价，官网可下）。

- 赛题讲评入口：https://dxs.moe.gov.cn/zx/hd/sxjm/sxjmstjp/gjsbqgdxssxjmjsstjp.shtml
- 优秀论文展示：https://dxs.moe.gov.cn/zx/hd/sxjm/sxjmlw/qkt_sxjm_lw_lwzs.shtml
