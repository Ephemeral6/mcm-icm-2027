# 国赛近三年真题与题型（2023–2025）

> 全部 15 道题的题名、背景、问题设置均从官网压缩包直接解出核对（`真题2023/`、`真题2024/`、`真题2025/` 为官方原件全集，本次新增补全了 2023 年 D/E 题、以及 A/B/D 类题目的完整问题设置，此前版本这部分标"待核对"）。官方下载入口：mcm.edu.cn "历年竞赛赛题"栏目（1992–2025 全部免费）。

## 0. 组别与选题（对本科队最关键的一条）

- 竞赛设**本科组**（所有大学生可参赛）和**高职高专组**（仅高职高专及职业本科学生，也可选参加本科组），研究生不得参赛（2026 年第一次通知 + 章程原文）。
- 历年惯例：**本科组做 A/B/C 题，高职高专组做 D/E 题**（从历年获奖名单按题分组可印证，属经验推测而非官方明文规定，正式以 9 月选题界面为准）。
- **即：你们作为本科队，实际只在 A/B/C 三题里选**——比美赛六选一的选题压力小，也意味着下面 D/E 题（多为高职高专组）主要作为方法论参考，不是你们的备选题库。

## 2. 逐题详解

### A/B 题：偏机理，数理门槛高（本科组，不建议）

- **2023A 定日镜场的优化设计**：塔式光热电站定日镜阵列布局优化，给定坐标计算光学效率与输出热功率，优化镜面尺寸/安装高度/间距。纯几何+连续优化，涉及太阳位置计算与光路反射几何。
- **2023B 多波束测线问题**：海洋测深场景，根据换能器开角、水深计算条带覆盖宽度与重叠率，设计测线布局。几何优化，需处理海底地形起伏。
- **2024A "板凳龙"闹元宵**：223 节板凳首尾相连盘旋的运动学建模，求各节位置/速度随时间变化。纯运动学/微分几何，约束方程量大。
- **2024B 生产过程中的决策问题**：抽样检测方案设计（假设检验+置信度）+ 生产链各环节决策优化（检测/装配/拆解/调换）。统计+决策树组合，概念性强。
- **2025A 烟幕干扰弹的投放策略**：无人机投放烟幕弹掩护目标，求飞行方向/速度/投放点/起爆点使遮蔽时间最长。连续优化+三维几何+多无人机协同，计算密集。
- **2025B 碳化硅外延层厚度的确定**：根据红外干涉光谱数据反演外延层厚度，推导多光束干涉条件。信号处理+物理机理建模，专业背景门槛高。

### C 题：连续三年都是"真实数据+决策/优化"（本科组，重点研究对象）

- **2023C 蔬菜类商品的自动定价与补货决策**：给定 3 年销售流水、批发价格、损耗率数据，分析品类销售规律，给出补货量与定价策略以最大化收益。方法：时间序列/回归分析销售规律 + 成本加成定价模型 + 线性/非线性规划求最优补货量。
- **2024C 农作物的种植策略**：34 个地块、多种作物、多年份，在产量/成本/价格不确定条件下给出多年最优种植方案，需交 Excel 结果文件。大规模整数/线性规划，约束条件多（轮作、豆类占比、地块面积），是国赛里工程量最大的题型之一。
- **2025C NIPT 的时点选择与胎儿的异常判定**：给定孕妇 BMI、孕周与胎儿染色体浓度数据，建立相关性模型、按 BMI 分组确定最佳检测时点。回归分析+分组优化+风险量化，医学背景但方法论是标准"数据驱动决策"。

### D/E 题：多为高职高专组，仅作方法论参考

- **2023D 圈养湖羊的空间利用率**：养殖场排产问题，给定羊栏容量约束和繁育周期，排产使利用率最大化。整数规划/排产调度。
- **2023E 黄河水沙监测数据分析**：给定 6 年水位/流量/含沙量数据，建模含沙量关系、分析突变性/季节性/周期性、预测趋势、设计采样方案。时间序列分析标准范式，可与 [12/02-预测类](../12-模型与算法策略/02-预测类.md) 对照学习。
- **2024D 反潜航空深弹命中概率问题**：潜艇位置服从给定分布（正态/截尾正态），求深弹命中概率。纯概率统计推导，几乎不需数据处理。
- **2024E 交通流量管控**：给定车牌识别车流数据（5 周、12 个交叉口），估计分时段车流量、优化信号灯配置、识别巡游车辆、评估管控效果。真实大规模数据+优化+效果评估，方法论上很像 C 题但归属高职高专组。
- **2025D 矿井突水水流漫延模型与逃生方案**：给定巷道网络（三维图结构），模拟突水后水流漫延、规划逃生路线。图论建模+流体仿真，可对照 [12/06-图论网络与仿真](../12-模型与算法策略/06-图论网络与仿真.md)。
- **2025E AI 辅助智能体测**：给定立定跳远运动员的人体姿态关键点视频数据（33 个关键点逐帧坐标），识别起跳/落地时刻、分析成绩影响因素、预测成绩。结构数据的特征工程+回归/分类预测，数据形式新颖，方法论上和美赛 C 题相似度最高——虽然是高职高专组题目，**仍推荐作为本科队的额外练习素材**（数据来源新颖、贴近"AI 生成数据+数学建模"的趋势）。

## 3. 题型规律与选题预判

1. **A/B 题偏机理**：几何、物理、运动学场景，数学功底要求高——与美赛 A/B 定位相似，不适合你们。
2. **C 题连续三年都是"真实数据 + 决策/优化"**：蔬菜定价（2023）→ 种植策略（2024）→ NIPT 时点（2025）。与美赛 C 题的区别：国赛 C 题的"优化/决策"味更重——美赛 C 更多是"预测+解释"，国赛 C 通常要在数据分析之上再解一个明确的优化/决策问题。
3. **对你们的结论：国赛选 C 题**，与美赛 C 题备赛高度复用（数据清洗、EDA、预测、不确定性），需要专项补的是**规划求解**（线性/整数规划建模 + Python 求解器，见 [12/03-优化类](../12-模型与算法策略/03-优化类.md)）——这是国赛 C 题比美赛 C 题多出的能力要求。
4. 练兵素材优先级：2025C（NIPT，数据已入库）→ 2024C（种植策略，数据已入库）→ 2023C（蔬菜定价，官网可下）→ 2025E（AI 体测，作为额外练习，虽属高职高专组）。
