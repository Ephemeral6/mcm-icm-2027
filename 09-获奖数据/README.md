## 3. 三个最重要的结论

### ① C 题获奖率并不比别的题高，选它是选"赛道宽"不是选"概率高"
O 奖率约 **千分之 1.4**；M 以上约 7–8.5%，与全赛平均（2025 MCM 全题 M 以上约 8%）基本持平。C 题 O 奖**数量**多（18–19 支 vs 其他题 4–7 支）纯粹因为分母大。选 C 的真正理由仍是：自带数据、方法模板化、学习材料最多——而不是获奖更容易。

### ② ⚠️ C 题是取消资格重灾区
2025 年全赛 514 个 Disqualified 中约 **77%（394 个）集中在 C 题**，C 题 DQ 率 3.1%，显著高于全赛平均 2%。原因构成官方未公布（推测与查重/AI 未披露相关，未验证），但对策明确：
- AI 使用 100% 留痕、100% 披露（见 02）；
- 赛前退出所有数模讨论群，杜绝思路撞车；
- 参考文献逐条核实，杜绝 AI 编造文献。
**你们队最大的可控风险就是这个。**

### ③ C 题 O 奖几乎被中国高校包揽，且不是名校专属
- 2025 年 18 支 O 奖中 **17 支中国高校**：安徽师范大学、暨南大学、四川大学、南京理工大学、三峡大学、南方科技大学、重庆大学、东北财经大学、西安交通大学×2、国防科技大学、上海交通大学、东北大学、厦门大学、中国石油大学(北京)克拉玛依、郑州大学、南京农业大学（唯一例外：美国 Stuyvesant 高中）。
- 2026 年 19 支中 **18 支中国高校**（外交学院、中南大学、苏州大学、西南大学、深圳大学等；唯一例外：威斯康星大学麦迪逊分校）。
- 名单里有大量双非院校——**打法对了，普通院校完全能拿 O 奖**。
