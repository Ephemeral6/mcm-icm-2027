## 2. 真题与获奖论文获取

- **官方**：真题免费（上表链接）；O 奖论文全文由 COMAP 出版（UMAP Journal）。
- **GitHub**：`2004-2020 MCM/ICM Outstanding Papers` 合集（在 `HuangCongQing/Algorithms_MathModels` README 中有引用链接）；2021 年后的 O 奖论文多在知乎/公众号流传，搜"美赛 XXXX 年 C 题 O奖论文"。
- **微信公众号**（中文社区资料入口，未验证质量）：「数学模型」（历年优秀论文汇总）、「双愚」（回复"数学建模资料"取备赛包）。
