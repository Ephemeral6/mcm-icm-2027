# 2027 美赛（MCM/ICM）备赛仓库

备战 2027 年 1 月底的 MCM/ICM（美国大学生数学建模竞赛，COMAP 主办）。三人队，从零开始，主攻 C 题（数据类）。这个仓库存放备赛调研的全部结果和资料，方便随时回顾。

## 赛制要点

- 六题选一：A 连续、B 离散、C 数据洞察、D 运筹/网络、E 可持续性、F 政策。A–C 属 MCM，D–F 属 ICM。
- 每队最多 3 人，须同校在读（本科或高中，研究生不行），专业不限。指导教师可以是本校任何教职工。
- 没有中国区预选：中国队伍直接在 COMAP 官网报名，和全球队伍走同一条路。高校的"校内选拔"是学校自设的报销和名额管理机制，不是官方要求；小美赛、亚太赛与 COMAP 无关，不是美赛前置。详见 [中国区参赛路径](01-赛事全貌与报名/中国区参赛路径.md)。
- 比赛 96 小时开卷，可查资料，但不能和队外任何人讨论（包括指导教师）。盲评，论文里不能出现姓名和学校。
- 论文为英文 PDF，上限 25 页，摘要页、目录、参考文献、附录、代码都算在内。摘要页必须是第一页。
- AI 政策：允许用 ChatGPT/Claude 等，但要在正文行内引用、文末附 "Report on Use of AI Tools"（不占页数，需列出每次的提问和完整回答）、提交表单里申报。不披露会被按抄袭处理，直接取消资格。
- 奖项从低到高：S（成功参赛）、H（约 22–23%）、M（约 6–7%）、F（约 1%）、O（约 0.14%）。

## C 题的实际数据（统计自官方 Results PDF）

| 指标 | 2025 年 C 题 | 2026 年 C 题 |
|---|---|---|
| 选题队数 | 12,573（占 MCM 的 60%） | 13,185（占六题的 41%） |
| O 奖 | 18（0.14%） | 19（0.14%） |
| M 奖及以上 | 约 7.3% | 约 8.5% |
| H 奖及以上 | 约 29.7% | 约 30.5% |
| 取消资格 | 394（3.1%，占当年全赛 DQ 的 77%） | 159（1%） |
| O 奖中的中国高校 | 17/18 | 18/19 |

从数据看：C 题获奖率并不比其他题高，选它的理由是数据现成、方法成熟、参考材料多，而不是更容易得奖；C 题被取消资格的比例明显偏高，AI 使用留痕和披露是头号风控；O 奖名单里有不少双非院校，说明打法比出身重要。

## 调研方法说明

调研分三轮（2026-07-12）：第一轮是赛制、AI 政策、题型、分工、路线、评审的全景；第二轮专门查 C 题——从官方 Results PDF 逐行统计获奖数据，并找到一篇 2024 年 C 题 O 奖论文核对了它的 AI 披露写法；第三轮整理模型选用策略。关键结论都经过多来源交叉验证，文档中统一标注两个可信度等级：

- 【官方·已验证】：COMAP 一手材料，多方核对
- 【经验·未验证】：知乎/CSDN 等社区内容，仅供参考

规则类信息一律以官方为准。调研中发现并排除了几条流传较广的错误说法（题目提前发布、官方免费公开 O 奖论文全文、20 页页数限制、熵权法完全客观等），分散记录在各板块中。
