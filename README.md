# 2027 美赛（MCM/ICM）备赛仓库

面向 **2027 年 1 月底 MCM/ICM**（美国大学生数学建模竞赛，COMAP 主办）的完整备赛调研与资料留痕。

**队伍画像**：中国高校三人队，从零起步；核心优势是 AI 工具使用能力；主攻方向 **C 题（数据洞察）**。

### 赛制一页纸

- 六题任选一：**A** 连续 / **B** 离散 / **C** 数据洞察 / **D** 运筹·网络 / **E** 可持续性 / **F** 政策（A–C 为 MCM，D–F 为 ICM）
- 每队 ≤3 人同校在读（本科或高中，研究生不可），任何专业；指导教师可为本校任何教职工
- 开卷 96 小时，但**严禁与队外任何人讨论**（含指导教师）；全文盲评（不得出现姓名/校名）
- 论文：英文 PDF，**25 页上限**（摘要页/目录/参考文献/附录/代码全计入），摘要页必须是第一页
- **AI 政策**：允许用 ChatGPT/Claude 等，但须行内引用 + 附 "Report on Use of AI Tools"（不计页数，逐条 Query/Output）+ 提交表单申报；**未披露按抄袭处理、取消资格**
- 奖级（低→高）：S（成功参赛）→ H（约 22–23%）→ M（约 6–7%）→ F（约 1%）→ O（约 0.14%）

### 用官方数据校准预期（第二轮调研，逐行统计官方 Results PDF）

| 指标 | 2025 年 C 题 | 2026 年 C 题 |
|---|---|---|
| 选题队数 | 12,573（占 MCM 60%） | 13,185（占六题 40.9%） |
| O 奖 | 18（0.14%） | 19（0.144%） |
| M 奖及以上 | ≈7.3% | ≈8.5% |
| H 奖及以上 | ≈29.7% | ≈30.5% |
| **取消资格 DQ** | **394（3.1%，占全赛 DQ 的 77%）** | 159（1%） |
| O 奖中中国高校 | 17/18 | 18/19 |

三个结论：**① C 题获奖率不比别的题高**（选它是选赛道宽、材料多）；**② C 题是 DQ 重灾区**（AI 留痕披露是第一风控）；**③ O 奖名单里大量双非院校**（安徽师大、三峡大学、郑州大学……打法对了够得着）。

## 🔍 调研方法与可信度

- 两轮深度调研（2026-07-12），各 100+ 检索/抓取代理并行工作：
  - **第一轮**：赛制/AI 政策/题型/分工/路线/评审全景。抓取 23 个信源，25 条关键论断做 3 票交叉验证（24 确认、1 否决）。
  - **第二轮**：C 题专项实证。验证代理直接下载 COMAP 官方 Results PDF 用 pdftotext 逐行计数复现获奖数据；实测下载 O 奖论文全文并核验其 AI Use Report 原文。
- 全部文档统一标注：**【官方·已验证】**（COMAP 一手来源 + 3 票确认）/ **【经验·未验证】**（知乎/CSDN 社区经验，仅供参考）。规则类信息一律以官方为准。
- 被证伪的流传说法：题目提前 10 分钟发布（❌）、官方免费公开 O 奖论文全文（❌）、20 页页数限制（❌ 过时，现行 25 页）。
