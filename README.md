## 第二轮调研的三个颠覆性发现

1. **C 题获奖率并不更高**：O 奖率仅 0.14%，M 以上约 7–8.5% 与全赛持平——C 题 O 奖数量多纯因分母大（2025 年占 MCM 60%、2026 年占全赛 41%）。选 C 是选"赛道宽、材料多"，不是选"概率高"。
2. **C 题是取消资格重灾区**：2025 年全赛 514 个 DQ 中 77%（394 个）在 C 题，DQ 率 3.1%。AI 留痕披露 + 赛前退群 + 核实文献，是你们最重要的风控。
3. **O 奖不是名校专属**：2025/2026 两年 C 题 O 奖 37 支中 35 支是中国高校，含大量双非院校（安徽师大、三峡大学、郑州大学、南京农业大学……）。打法对了完全够得着。
