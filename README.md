2027 年美赛于 **北京时间 2027 年 1 月 29 日（周五）早 6:00 开赛，2 月 2 日（周二）早 9:00 结束**（96 小时）。COMAP 官方明确允许使用生成式 AI（ChatGPT/Claude 等），但必须严格披露——这对你们"AI 强、数理一般"的队伍是最大机会，也是最大红线。选题方向初步锁定 **C 题（数据洞察）**，2025 年 C 题产出 18 支 O 奖队、远超其他题。
