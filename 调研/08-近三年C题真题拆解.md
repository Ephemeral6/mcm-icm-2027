## 2. 2026 年 C 题详拆（官方·已验证，最具参考价值）

**背景**：美国真人秀 Dancing with the Stars（DWTS），选手每周由评委打分 + 观众投票共同决定去留，但**粉丝票数从不公开**。

**数据**：仅一个文件 `2026_MCM_Problem_C_Data.csv`——第 1–34 季选手信息、名次、逐周评委打分（`weekX_judgeY_score`，1–10 分；淘汰后记 0；缺失记 N/A；一周跳多支舞时分数可能为平均后的小数）。

**任务链**（每问依赖上一问）：
1. **核心建模**：估计每周未公开的粉丝票数，并给出一致性度量与不确定性度量——这是全题的地基；
2. 用估计的票数对比节目用过的两种合成计分法：rank 法（S1–2 及推测 S28–34）vs percent 法（S3–27），分析两法是否导致不同淘汰结果；
3. 分析四个著名争议案例：Jerry Rice、Billy Ray Cyrus、Bristol Palin、Bobby Bones（低分选手靠粉丝票走远）；
4. 评估 S28 起引入的"评委在垫底两组中选淘汰者"新机制的影响；
5. 推荐未来采用哪种方案，并**设计一个更公平的新计票系统**；
6. 给 DWTS 制作人写 1–2 页 memo。

**格式要求**（题面原文）：总 25 页内含摘要页、目录、完整解答、memo、参考文献；AI Use Report 附在 PDF 末尾、不计页数。

**解题要点提示**（我的分析，供训练用）：粉丝票是隐变量 → 反演/贝叶斯估计/约束优化思路；"一致性+不确定性度量"是官方明示的采分点；公平性定义本身就是建模对象。

## 3. 2024 年 C 题（经 O 奖论文侧面还原，部分验证）

- 用温网 2023 男单逐分数据量化"momentum（势头）"，判断势头是否存在（驳斥"纯运气"说）、预测比赛流向转折、给教练写建议。
- O 奖论文 2406324 的路线见 [10-O奖论文实证](10-O奖论文实证与AI披露模板.md)。
- 评委事后态度（第一轮已调研，社区转述）：批评"只堆机器学习"，赞赏"查专业资料识别构成势头的要素→构建可解释的动态表达式"。

## 5. 对 2027 备赛的启示

1. C 题连续三年 = "真实数据 + 隐变量估计/预测 + 公平性/机制评价 + 非技术写作"，模拟训练直接用 2026 年题（数据免费可下）。
2. 不确定性量化（置信区间、敏感性分析）连续被题面点名，必须成为队伍的标配技能。
3. 每年都要求给非技术受众写 1–2 页 memo/信——写作手要专门练这个文体。
4. 真题与数据全部免费：`contest.comap.com/undergraduate/contests/mcm/contests/{年份}/problems/`（1999–2026 全部可下，已实测）。
